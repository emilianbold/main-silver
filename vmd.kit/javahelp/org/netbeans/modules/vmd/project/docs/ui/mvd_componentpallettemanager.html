MIDP Component Palette Manager

See Also

Use the MIDP Component Palette Manager to add custom components the Visual Mobile Designer (VMD) Component Palette. After you add custom components to the Palette, you can drag and drop the components into the Flow Designer and Screen Designer as you design your application.

You open the Component Palette Manager by right-clicking in the Component Palette and choosing Palette Manager.

The Components window displays all components available for your project. The menu tree is organized in the same categories as the Component Manager.

  • The components available are determined by the MIDP version supported in the active project.

Click the following buttons to add or manipulate components:

  • Add from Project. Opens a wizard for adding new displayable components and items.
  • New Category. Brings up a dialog that allows a new category to be added to the palette.
  • Move Up. Moves a component up on the menu tree.
  • Move Down. Moves a component down on the menu tree.
  • Remove. Removes the selected custom category or item.
