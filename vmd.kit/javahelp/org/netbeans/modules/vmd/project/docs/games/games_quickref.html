Game Builder: Quick Reference

This topic describes common tasks you can perform in the Game Builder:

To perform this task Follow these steps
Create a game using the game design file template
  1. Choose File > New File. This opens the New File wizard.
  2. Under Categories, choose MIDP
  3. Under File Types choose Visual Game Design.
  4. Choose a name for your game's MIDP Class Name.
  5. Choose a source package and Finish.
Add a new scene
  1. To add another game level click the New Scene button on the scene editor toolbar.
  2. Name the scene and hit the OK button. This creates an empty scene that contains no layers.
  3. Add layers from existing sources by right clicking the game canvas and choosing Add Tiled Layer - tiledlayername.
Add a new tiled layer to a scene
  1. To add a new tiled layer to a scene, right click inside the scene editor and select Create tiled layer.
  2. Give the new tiled layer a filename and choose an image form the Select Image list or click the Import Sample Images button to add to your image library for the game builder.
  3. Drag the tiled layer to the desired position on the scene. Repeat this process until the scene is filled with the desired background visuals for your game. Remember that Sprites must remain on top of the tiled layers to be visible.
Add a sprite
  1. Click the New Sprite button on the scene editor toolbar.
  2. Select an image or set of images from the Create New Sprite Dialog and give the sprite a name.
How to import sprites and background layers.
  1. Open up the projectname directory and navigate to the /src/res folder and add png or other suitable image types there. You can also click the Import Sample Images button in the Create Sprite and Create Tiled Layer dialogs boxes.
