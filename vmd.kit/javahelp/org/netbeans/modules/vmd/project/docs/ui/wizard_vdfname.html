New Visual Mobile Designer File Wizard:
Name and Location

See Also

You open the New MIDP Visual Design File wizard by choosing File > New File and selecting a Visual Design file () from the MIDP category. You use this wizard to open a visual file, which allows you to create or modify your MIDP project using the Visual Mobile Designer.

In the Name and Location page, you set the following properties:

  • MIDlet Name. The displayable name that users see when the application is run on a mobile device.
  • MIDP Class Name. The name of the new MIDP class. Note that there cannot be two classes with identical names within the same project.
  • MIDlet Icon. An icon that is associated with the MIDlet. Users see this icon when the MIDlet is run on a mobile device.
  • Project. Shows the name of the project selected in the previous step. this field is not editable.
  • Package. Specifies the location of the MIDlet class. You can select an existing package from the Combo Box, or type in the name of a new package. The new package is created along with the class.
  • Created File. Shows the name and location of the file when it is created. This field is not editable.
  • Choose MIDP version. Sets the MIDP version of the components that are available in the Visual Mobile Designer. You can choose between versions 1.0 and 2.0.
  • When a new MIDlet is created, the IDE automatically adds the MIDlet to the Application Descriptor file.
