Upgrading CLDC/MIDP Projects from Version 5.0 to 6.0

See Also

This dialog appears when you open the NetBeans 6.0 IDE with NetBeans Mobility pack 6.0 and import MIDP/CLDC projects created using versions 5.0 or earlier of the Mobility Pack.

Because of the following changes, projects imported and saved into the NetBeans 5.5 or 6.0 IDE cannot be re-opened in previous versions of the IDE:

  • Changes in how pre- and post-init user code is written.
  • Implementation of multi-guarded blocks into which you can write code directly.
  • Changes in the WaitScreen and SplashScreen Java ME implementations.

If you do not want to import and update project, close the project without saving it.
