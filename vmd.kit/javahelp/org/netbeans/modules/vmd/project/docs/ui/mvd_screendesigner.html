Screen Designer

See Also

The Screen Designer enables you to customize screens that have been added to the flow of your application in the Flow Designer window.

The Screen Designer window opens when you double-click on a Screen icon in the Flow Designer window. The Screen Designer has two main sections:

  • Device Screen
  • Assigned Resources

Device Screen

The Device Screen simulates how the component appears on a real device. All components appear in the order they are listed in the Navigator view.

New components can be dragged and dropped directly to the desired location of the Device Screen. Only components that are valid for the screen type can be added.

Assigned Resources

This portion of the Screen Designer shows all the assigned commands, assigned item commands, and resources attached to the screen. You can select the box to display and edit the properties of the Transaction Source in the Properties view.
