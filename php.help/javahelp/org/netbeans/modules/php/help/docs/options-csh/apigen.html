PHP ApiGen Options

See Also 

  • ApiGen script: The path to the ApiGen run script.
  • Search. The IDE searches for all ApiGen run scripts that you have installed. A dialog opens in which you select which run script to use.
  • Browse... Manually browse your file system for the run script.

Note: ApiGen is not installed with NetBeans IDE. Click the first Learn More link to open a browser window with ApiGen's installation instructions.

You can specify ApiGen run parameters either by providing individual parameters via the command line or by using a configuration file. The IDE lets you specify a configuration file per PHP project, in the project's Properties. Note that properties specified on the command line override properties in a configuration file.
