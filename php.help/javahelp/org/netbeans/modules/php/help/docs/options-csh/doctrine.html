PHP Options: Doctrine2

  • Doctrine2 script: The path to the Doctrine2 run script.
  • Search. The IDE searches for all Doctrine2 run scripts that you have installed. A dialog opens in which you select which run script to use.
  • Browse... Manually browse your file system for the run script.

Note: Doctrine2 is not installed with NetBeans IDE. Click the first Learn More link to open a browser window with Doctrine2's installation instructions.

You can specify configuration options in the Doctrine2 script field. The contents of this field are run on the command line when you launch Doctrine2 from the IDE.
