Project Properties: PHPDoc

In this dialog you set project properties for documentation generated by PHP Documentor. You must have PHP Documentor installed. You must configure NetBeans IDE for PHP to use PHP Documentor, in Tools > Options > PHP > PHPDoc (On MacOS, this is NetBeans Preferences > PHP > PHPDoc).

  • Target Directory: The directory containing the generated PHP Documentor documentation.
    • Browse... Browse your file system for a target directory.
  • Title: The title of the generated documentation.
