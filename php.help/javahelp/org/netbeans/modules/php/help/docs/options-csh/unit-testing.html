PHP Unit Testing Options

  • PHPUnit script: The path to the PHPUnit run script.
  • Search. The IDE searches for all PHPUnit run scripts that you have installed. A dialog opens in which you select which run script to use.
  • Browse. Manually browse your file system for the run script.

Note: PHPUnit is not installed with NetBeans IDE. Click the Learn More link to open a browser window with PHPUnit's installation instructions.
