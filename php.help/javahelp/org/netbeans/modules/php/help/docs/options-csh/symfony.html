Symfony Options

  • Symfony script: The path to the Symfony run script and default parameters.
  • Search. The IDE searches for all Symfony run scripts that you have installed. A dialog opens in which you select which run script to use.
  • Browse... Manually browse your file system for the run script.

Note: Symfony projects contain a Symfony run script. The run script you specify in this page is only used for new projects or in case the project's run script is not valid.

You can specify default command line options for running Symfony projects or applications.
