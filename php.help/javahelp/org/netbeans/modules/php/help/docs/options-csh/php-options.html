PHP Options

On this page, you set paths used by the NetBeans IDE PHP editor.

Tabs

Select the tab for the options you want to set. Note that you only need to set the options for a feature if you use that feature.

General

Select PHP Interpreter, set debugging options, and set global include path.

Unit Testing

Set path to PHPUnit script.

PHPDoc

Set path to PHPDoc script. Include command-line options.

Symfony

Set the path to the Symfony framework run script. Also set default parameters for projects and applications.

Zend

Set the path to the Zend Framework run script. Register the provider if using Zend Framework 1.10 or newer.

Exporting and Importing Options

Press Export to create a ZIP archive of NetBeans IDE options. A dialog opens in which you choose which options to export. This function applies to all options, not only PHP options.

Press Import to import a ZIP archive of NetBeans IDE options.
