Symfony2 Options

  • Symfony: The path to the Symfony Standard Edition .zip archive. If you do not have a copy of Symfony Standard Edition, click the link Download Symfony Standard Edition.
    • Search. The IDE searches for all Symfony run scripts that you have installed. A dialog opens in which you select which run script to use.
    • Browse... Manually browse your file system for the run script.
    • Symfony projects contain a Symfony run script. The run script you specify in this page is only used for new projects or in case the project's run script is not valid.
  • Ignore cache directory by default. Deselect this for NetBeans IDE to include the /cache directory when scanning for Symfony functions. This can be useful if you cache actions.
    • This box sets globally whether cache directories are ignored. You can set whether the cache directory is ignored for an individual project in that project's Properties, under Frameworks > Symfony.
