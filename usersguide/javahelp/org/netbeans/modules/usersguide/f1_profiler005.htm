Select Profiling Task: New Custom Configuration Dialog Box

Use to create a named set of options for a profiling task.

Element Description

Name

 Specifies the name of the new custom configuration.

Initial Settings

 Specifies a configuration on which the new configuration is to be based.

Select Default if you want the new custom configuration to be based on the default configuration for the profiling task.

Select From Existing Configuration to select a configuration from the list.
