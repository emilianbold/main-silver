Project Configuration Manager

Use to add a new project configuration to a project or remove project configurations from a project.

Element Description

Configurations

Displays a list of project configurations.

Add

Opens the Add Project Configuration dialog, where you can enter a name for a new project configuration.

The new configuration name is added to the list of available configurations. The new configuration has the same properties as DefaultConfiguration.

Add More

Creates additional configurations from the listed templates.

Check or uncheck a device to include or exclude it from the configuration.

Duplicate

Opens the Duplicate Project Configuration dialog, where you can enter a name for a new project configuration. The new configuration name is added to the list of available configurations. The new configuration has the same properties as the configuration you are duplicating.

Rename

Specifies a new name for the selected configuration.

Remove

Deletes the selected configuration.

Save As Template

Saves the selected project configuration as a template. You can then use the template to create other project configurations in the Add Project Configuration dialog box.
