CSS Styles Window

See Also 

The CSS Styles window enables you to edit the declarations of rules for HTML elements and selectors in a CSS file. You can open the CSS Styles window by choosing Window > Web > CSS Styles from the main menu.

The upper pane of the CSS Styles window displays a list of the rules that are defined in the CSS file. You can filter the list by typing a string in the text field at the top of the pane. You can double-click an rule in the list to navigate to the line in the CSS file where the rule is defined.

The lower pane of the CSS Styles window displays a list of the properties and values that are specified in the declarations of a rule. In the editor, when your insert cursor is in a line that contains a declaration for a CSS rule the properties for that rule are listed in the lower pane of the window. You can use the lower pane to create and delete rules and to modify, add and remove the properties of declarations. You can use the dropdown list to select values for each property or type a value in the text field.

The toolbar in the lower pane contains the following toggle buttons that you can use to modify how the properties are displayed.

Button Description
Only the properties that have a specific value are displayed.
Show all properties, organized by category.
Show all properties, listed alphabetically.

You can click the Menu icon () to display a menu with the following options.

  • Add Property. Opens a dialog box that enables you to add properties to the selected rule that is displayed in the lower pane of the window.
  • Create Rule. Opens a dialog box that enables you to create a new rule in a style sheet.
  • Delete Rule. Deletes the selected rule that is displayed in the lower pane of the window. This option is disabled if no rule is selected in the style sheet.
