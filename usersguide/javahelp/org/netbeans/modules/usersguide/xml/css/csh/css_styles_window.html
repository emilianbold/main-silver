CSS Styles Window

See Also 

The CSS Styles window enables you to edit the properties of HTML elements and selectors in a CSS file. You can open the CSS Styles window by choosing Window > Web > CSS Styles from the main menu.

The upper pane of the CSS Styles window displays a list of the elements that are defined in the CSS file. You can filter the list by typing a string in the textfield at the top of the pane. You can double-click an element in the list to navigate to the line in the CSS file where the element is defined.

When your insert cursor is in an element definition in the CSS file in the editor, the properties of that element are listed in the lower pane of the CSS Styles window. You can use the lower pane to create rules for elements and to modify, add and remove the properties of elements. You can use the dropdown list to select values for each property.

The toolbar of the lower pane contains the following options.

You can use the buttons in the toolbar to modify the how the properties are displayed. The following table lists the buttons that are available in the toolbar when a CSS file is open.

Icon Description
Show All CSS Properties.
Show Property Categories.
Sort properties by natural order.
Sort properties alphabetically.

You can click the Menu icon () to display a menu with the following options.

  • Add Property.
  • Create Rule.
  • Delete Rule.
