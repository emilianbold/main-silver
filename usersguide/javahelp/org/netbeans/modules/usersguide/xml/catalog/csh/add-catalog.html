Add Catalog Dialog

Use this dialog to add an existing XML Schema and DTD Catalog to your project. For information about creating an XML Schema and DTD Catalog, see Creating and Editing an XML Catalog.

This dialog opens when you click Add Catalog in the DTDs and XML Schemas dialog. To open the DTDs and XML Schemas dialog, go to Tools > DTDs and XML Schemas. See Registering a Local DTD or XML Schema Resource.

The Add Catalog dialog contains the following fields:

  • Catalog Type: Select the catalog type from the drop-down box. See XML Catalog Formats for more information.
  • Catalog URL: Location of the existing DTD or XML Schema catalog that you want to add to the project.
    • Browse: Browse for the catalog in your local file system.
  • Prefer Public ID: Select if the catalog resolver should try to use the PUBLIC identifier before resorting to the SYSTEM identifier. Selected by default.
