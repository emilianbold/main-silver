Standard Web Project Properties Dialog Box: Disabled JAX-RPC Web Services

You open the Standard Web Project Properties dialog box by right-clicking any and choosing Properties.

The JAX-RPC Web Services page lets you preconfigure the features that are used by the wscompile tool to compile a JAX-RPC web service:

If you have not created any JAX-RPC web services, this page is empty. To create JAX-RPC web services, see About Implementing Web Services.

This page is for JAX-RPC (J2EE 1.4) web services only. Even though the project does not contain JAX-RPC web services, it could contain JAX-WS (Java EE 5) web services.
