Issue Tracking Tasks: Quick Reference

The following table describes common issue tracking tasks.

To perform this task Follow these steps
Add an issue tracker
  • Choose Window > Services from the main menu.
  • Right-click the Issue Trackers node and choose Create Issue Tracker.
  • Specify the connection details. Click OK.
Find open issues
  1. From the main menu, choose Team > Find Issues.
  2. Select the issue tracker for the project.
  3. Specify the search criteria.
  4. Click Search.
Report an issue
  1. From the main menu, choose Team > Report an Issue.
  2. Select the issue tracker for the project.
  3. Specify the details of the issue.
  4. Click Submit Issue.
Update issue
  1. Find the issue that you want to update.
  2. Open the form for the issue.
  3. Add comments or other issue details.
  4. Click Submit Changes
Close an issue
  1. Find the issue that you want to resolve.
  2. Open the form for the issue.
  3. Select Resolve from the Status drop-down list.
  4. Select how the issue is resolved from the Resolution drop-down list.
  5. Add comments or other issue details.
  6. Click Submit Changes.
Mark an issue as a duplicate
  1. Find the issue that you want to resolve.
  2. Open the form for the issue.
  3. Select Resolve from the Status drop down list.
  4. Select Duplicate from the Resolution drop down list.
  5. Click the Select button next to the Of Issue text field.
  6. Specify the issue in the dialog box. Click Select.
  7. Add comments or other issue details.
  8. Click Submit Changes.
Apply attached diff patch
  1. Open the form for the issue that contains the attached patch.
  2. Click "apply" next to the attachment.
  3. Locate the file or directory where you want to apply the patch. Click Apply.
Add an issue to the Action Items List
  1. Open the form for the issue.
  2. Click "Add to Action Items" in the top of the issue tab.
