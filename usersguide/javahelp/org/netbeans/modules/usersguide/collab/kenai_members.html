Chatting with Project Members

See Also

A Kenai project can have multiple members collaborating on a project. If the chat room for a Kenai project is enabled, you can exchange messages in the project's chat room from within the IDE. You can also engage in private chats with members.

To chat with project members you must be logged in to Kenai and logged in to the chat server. You can log in to Chat when you log in to Kenai. If you do not login to Chat when you log in to Kenai, you can enable Chat after you login to Kenai by clicking the Offline icon in the status bar. For more about enabling and disabling Chat, see Logging in to Chat.

Project Chat Rooms

You can use the chat window for the project to exchange messages with other project members. Each Kenai project can have an associated chat room open to members of the project.

To open the project's chat window:

  • Choose Window > Chat from the main menu.
  • Expand the dropdown list and select the project.

Alternatively, you can click a project's Message icon (  ) in the Kenai dashboard to open the Chat window

Private Chats

You can have private chats with individual project members from within the IDE. To initiate a private chat, double-click the name of a project member to open a chat window.

Sending Links and Files

You can use the Insert Link button (  ) in the chat window to quickly send a link to a file or issue as a message. You can open the dropdown list for the button to choose from open files or issues.

Note.

  • To open a link to a file that you receive, the referenced project must be open in the IDE.
