Chatting with Project Members

See Also

A Kenai project can have multiple members collaborating on a project. If the chat room for a Kenai project is enabled, you can exchange messages in the project's chat room from within the IDE. You can also engage in private chats with members of projects.

To chat with project members you must be logged in to Kenai and logged in to the chat server. You can log in to Chat when you log in to Kenai. If you do not log in to Chat when you log in to Kenai, you can enable Chat after you login to Kenai by clicking the Offline icon in the status bar. For more about enabling and disabling Chat, see Logging in to Chat.

Project Chat Rooms and Private Chats

You can use the Chat window for a project to exchange messages with other project members. Each Kenai project can have an associated chat room open to members of the project. You can also have private chats with individual project members from within the IDE.

To enter a chat room or initiate a private chat, perform the following steps.

  1. Choose Window > Chat from the main menu.
  2. Click the New Chat button (  ).

    The window displays a list of project chat rooms and project members that are online and available for chat.

  3. Double-click the name of the project to enter the project chat room, or the name of the member to start a private chat.

    You can use the drop-down list to filter the list of online members according to project.

Notes.

  • Alternatively, you can click a project's Message icon (  ) in the Team Dashboard to open the chat room.
  • You can also start private chats with a project member by double-clicking the Member Online icon (  ) when it is displayed in other windows (for example, the Team Dashboard, an issue form).

Sending Links and Files

You can use the Insert Link button (  ) in the Chat window to quickly send a link to a file or issue as a message. You can open the drop-down list for the button to choose from a list of open files or issues.

Note.

  • To open a link to a file that you receive, the referenced project must be open in the IDE.
