Select Issue Keyword

See Also

You use this dialog box when you want to assign a registered keyword to an issue or search for issues that contain the selected keyword. You open this dialog box by clicking the Search button (...) next to the Keywords text field in an issue's form or in the Find Issues search form.

This dialog displays a list of possible keywords that can be assigned to the project's issues. The list of keywords is determined by the administrator of the project's issue tracker. Select the keyword or keywords that you want to add to the issue or query.
