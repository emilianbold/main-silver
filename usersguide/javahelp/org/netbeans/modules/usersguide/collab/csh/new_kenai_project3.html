New Kenai Project Wizard: Summary

See Also

The Kenai Project wizard enables you to create a new and share projects on Kenai. You open the wizard by choosing Kenai > New Kenai Project from the Team menu. You can share the sources of an open project by selecting the project in the Projects window and choosing choosing Kenai > Share Local Project on Kenai. Alternatively, you can right-click a project in the Projects window and choose Share on Kenai in the popup menu.

The Summary page enables you to view a summary of the Kenai services that will be set up when you click Finish in the wizard. Depending on the options you selected in the wizard, the Summary page displays the following details about the Kenai project.

  • the name and location of the project that will be created on Kenai.com
  • the location of the local repository
  • the location of the remote repository
  • the projects and/or files that will be committed to the remote repository
  • if an issue tracker for the project will be set up
  • if a chat room for the project will be created
  • If any of the details are incorrect, use the Back button to change the wizard options.

After you click finish, the IDE will create the project on Kenai.com. The project sources are committed to a repository and can be downloaded freely. However, only approved members of the project team can commit changes to the repository and participate in the project's chat room.
