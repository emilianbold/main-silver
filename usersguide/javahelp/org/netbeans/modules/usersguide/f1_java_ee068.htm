Visual Editor: Message Security

Use to specify the message security entries for the message-security-binding property when using Message Security Binding for a web service endpoint or port.

You also use this dialog box when configuring message security policy for a port for web service clients.

This dialog box enables you to edit the following properties:

  • Java Method Name (required)
  • Request Authorization Source
  • Request Authorization Recipient
  • Response Authorization Source
  • Response Authorization Recipient
