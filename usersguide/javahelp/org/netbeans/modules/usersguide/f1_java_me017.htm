Quick Project Run

Use to test how a your application performs on different mobile devices.

Right-click a project and choose Run With to open this dialog box.

Element Description

Select Platform

Specifies a platform on which to run the most current build of your project.

The IDE runs your application on the selected platform, but does not build the project or create a distribution JAR for that platform.

Select Device

Specifies a device in which to run he most current build of your project.
