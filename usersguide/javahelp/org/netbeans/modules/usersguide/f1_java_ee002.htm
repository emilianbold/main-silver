Enterprise Application Client Project Properties Dialog Box: Documenting

Use the Documenting panel to configure how the IDE generates Javadoc documentation for your classes. To open the Enterprise Application Client Project Properties dialog box, right-click any application client project node and choose Properties. Under the Build tree, choose Documenting.
