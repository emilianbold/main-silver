HTML Code Completion Options

See Also 

Set these options for HTML code completion in the Options > Editor > Code Completion tab, when HTML language is selected. These options apply globally to HTML, and only to HTML.

  • Auto Popup Completion Window. When selected, enables the editor to display a code completion box with options for how to complete an expression.
  • Equals-sign (=) Keypress After Attribute Name Autocompletes Quote Set If you type an HTML tag with an attribute name, like <img src, and then type =, the IDE adds a pair of quote tags and places your cursor between the quote tags. So if you type <img src=, the IDE expands this to <img src="|". Enabled by default.
  • Quote-mark (") Keypress After Attribute Name Autocompletes Closing Quote If your cursor is immediately before a closing quote in an attribute value and you type a quotation mark ", the IDE moves your cursor to after the closing quote instead of adding another quotation mark.

    If the option "Equals-sign (=) Keypress After Attribute Name Autocompletes Quote Set" is disabled and you type an attribute name, equals sign, and opening quote, the IDE also inserts the closing quote of the attribute value.

    This attribute is enabled by default.

  • Completion Offers End Tags After Less-than (<) Character If this is enabled, and you press less-than (<), code completion offers you closing tags at the top of the popup window. Disabled by default.
