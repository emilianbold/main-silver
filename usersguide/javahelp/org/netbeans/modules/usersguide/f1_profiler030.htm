Select Heap Dump to Compare Dialog Box

Use to select a heap dump that is associated with the current project or select a heap dump that is located on your local system. The heap dump that you select is compared to the heap dump in the Heap window.

Element Description

From project

Selects a heap dump from the list of heap dumps that are associated with the current project.

From file

Selects the heap dump that you want to compare to the current heap dump in the Heap window. Click Browse to locate the heap dump.
