Analyze Performance: Define New Profiling Root

Use to specify a method and the class that contains the method. You can also edit an existing root method.

Element Description

Class Name

 Specifies the name of the class that contains the method.

Method Name

 Specifies the name of the method.

Method VM Signature

 Specifies the signature of the method in VM format.
