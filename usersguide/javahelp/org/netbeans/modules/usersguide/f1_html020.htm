Options Window: HTML/JS: Grunt

You use this panel in the Options window to configure the IDE to use Grunt.

This panel contains the following elements.

Element Description

Grunt Path

 Specify the path to the Grunt cmd file.
    Install Grunt Click to display and install Grunt on your machine.
