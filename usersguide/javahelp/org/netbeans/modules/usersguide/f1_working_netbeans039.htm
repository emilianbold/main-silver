Plugins Manager: Installed

You use the Plugins manager to dynamically update the IDE from the registered update centers and to manage IDE plugins. You use the Installed tab of the Plugins manager to manage installed IDE plugins.

By default, the left pane displays a list of the activated features. When you select a plugin in the left pane a description of the feature is displayed in the right pane.

In addition to the list of plugins, this panel contains the following elements:

Element Description

Search

Type a string in the text field to filter the list of displayed plugins.

Activate

Click Activate to activate the selected plugin.

Deactivate

Click Deactivate to deactivate the selected plugin

Uninstall

Click Uninstall to remove the plugin from the list of plugins

The following table displays the icons used in the Active column to indicate the state of the feature or module.

Icon Description

The feature or plugin is installed and activated.

The IDE needs to be restarted to fully deactivate the plugin.

The plugin is installed but deactivated.

An installed plugin can be active or inactive. If a listed plugin is inactive, you may need to install additional plugins to use the plugin.
