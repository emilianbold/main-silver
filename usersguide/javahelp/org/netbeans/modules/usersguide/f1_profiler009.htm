Edit Global Filters Dialog Box

Use to edit, delete or modify global filters that are used in filter sets. You use filter sets to limit the classes that are profiled when analyzing application performance. How?

Element Description

New

 Creates a new row in the list and types the new name and values directly in the fields in the list.

Edit

 Modifies the selected filter. Click Edit and type directly in the field in the list to edit the name or value.

You can also edit a field by double-clicking in the field.

Delete

 Removes the selected filter from the list when you click Delete.

Move Up and Move Down

 Reorders the filter list when you click Move Up and Move Down on a selected filter.

The new order is also displayed in the Customize Filter Sets dialog box.
