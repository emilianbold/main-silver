Customize Toolbars Window

See Also

The Customize Toolbars window enables you to modify the contents of the IDE toolbars by adding or removing toolbar icons. You can also use the window to create a custom named toolbar configuration.

You open the window by choosing View > Toolbars > Customize from the main menu.

The Customize Toolbars window displays a list of IDE commands that are organized by category. You add commands to a toolbar in the IDE by dragging the icon for the command from the window to the desired position in a toolbar. You can remove commands from a toolbar by dragging the icon from the toolbar into the window.

You can click the New Toolbar button to open the New Toolbar dialog and create a new named toolbar configuration.
