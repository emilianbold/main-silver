Editor Options: Mark Occurrences

This tab enables you to specify which occurrences of Java or PHP keywords will be marked when the cursor is on one of their instances.

  • Language. Select Java or PHP.
  • Mark Occurrences Under Caret. Disable this to universally turn off the marking of keywords.
  • Specific Keywords (Java only). With Mark Occurrences enabled, select the categories of keywords whose occurrences you want to mark. Select Keep Marks for the marks to persist after you move the caret.
