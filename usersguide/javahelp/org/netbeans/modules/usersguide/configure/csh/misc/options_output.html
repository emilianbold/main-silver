Options Window: Miscellaneous: Output

See Also

You use this panel to specify the display options of the Output window.

You can access this panel by choosing Tools > Options from the main menu, clicking the Miscellaneous category and then clicking the Output tab.

You can set the following output options:

  • Output Window Settings. Click Restore to get the IDE's default settings for the Output window back.
    • Font. Set font details by clicking the ellipsis button to the right of the field.
      • Font Chooser. Specify the preferred font family, style, and size.
        • Font settings apply to tabs that are not wrapped only. Wrapped tabs always use a fixed-width font family.
    • Font size. Change the font size by using the scroll pane options.
    • Background Color. Specify the preferred background color.
    • Standard Color. Specify the standard text foreground color.
    • Error Color. Change the default color for the standard error foreground color.
    • Link Color. Select a color to be used as links foreground color.
    • Important Link Color. Select a color to be used as important links foreground color.
    • Link Style. By default, links are underlined. Specify the preferred link style.
  • Preview. Contains sample text that reflects changes you make in Output Window Settings.
