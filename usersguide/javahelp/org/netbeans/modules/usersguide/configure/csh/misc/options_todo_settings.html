Options Window: Miscellaneous: Action Items

See Also

You use this panel to specify the character patterns that are used for identifying action items. Action items are listed in the Action Items window.

You can access this panel by choosing Tools > Options from the main menu, clicking the Miscellaneous category and then clicking the Action Items tab.

You can set the following action item options:

  • ToDo Patterns. The text files displays the tag strings used in the source code to identify action items. Use the buttons to the right of the text field to add, delete or modify the ToDo patterns.
  • Show ToDos from comments only. If selected, ToDo patterns that are outside of comment blocks are ignored.
