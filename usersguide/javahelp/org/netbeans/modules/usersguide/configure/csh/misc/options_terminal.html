Options Window: Miscellaneous: Terminal

See Also

You use this panel to specify the display options of the command line Terminal window.

You can access this panel by choosing Tools > Options from the main menu, clicking the Miscellaneous category and then clicking the Terminal tab.

You can set the following Terminal options:

  • Font. Click the Browse button to select the display font.
  • Font Size.
  • Foreground Color. Select a font color from the list.
  • Background Color.
  • Selection Background Color.
  • History Size.
  • Tab Size..
  • Click To Type.
  • Scroll On Input.
  • Scroll on Output.
  • Wrap Lines.
  • Wrap Lines.
