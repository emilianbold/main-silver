Options Window: Miscellaneous: Terminal

See Also

You use this panel to specify the display options of the command line Terminal window. The Terminal window opens as a tab in the Output area of the IDE. You can open the Terminal by choosing Window > Output > Terminal from the main menu.

You can access this panel by choosing Tools > Options from the main menu, clicking the Miscellaneous category and then clicking the Terminal tab.

You can set the following Terminal options:

  • Font. Click the Browse button to select the display font.
  • Font Size. Use the arrows to set the size of the display font or type in the text field.
  • Foreground Color. Select a font color from the list.
  • Background Color. Select the background color of the terminal from the list.
  • Selection Background Color. Select the background color of selected text from the list.
  • History Size. Use the arrows to set the size of the terminal history or type in the text field.
  • Tab Size.. Use the arrows to set the default tab size or type in the text field.
  • Click To Type. When selected, you need to click the cursor in the terminal tab to give the terminal focus. When deselected, moving the cursor into the terminal tab will give the terminal focus.
  • Scroll On Input. When selected, the window will scroll to the bottom when you type at the prompt.
  • Scroll on Output. When selected, the window will scroll to the bottom when there is output in the terminal.
  • Wrap Lines. When selected, lines wider than the window will automatically wrap to the next line.
  • Ignore Global Shortcuts. When selected, global terminal shortcuts are ignored.

When you modify the Terminal options you can see the results of the change in the Preview panel.

You can return the Terminal options to the default state by clicking Restore.
