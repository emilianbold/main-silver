Editor Options: Spellchecker

Use this tab to specify various options related to the Source Editor spellchecker.

  • Use in. Select the file types for which you want to switch on the checking.
  • Dictionaries. Lists the dictionaries used for spell checking. You can add and remove dictionaries using the appropriate buttons.
    • A dictionary is a file with a list of words where every word is located on a new line. You need to specify the encoding of a dictionary file and for which locale it should be used when adding a dictionary via the Add Dictionary dialog box.
  • Default locale. Specify the locale to be used when there is no other locale is defined for the given dictionary.
