Options Window: General

See Also

You use this panel to set the IDE's web browser and to specify general proxy settings.

You can access this panel by choosing Tools > Options from the main menu, and then clicking the General category.

You can set the following general IDE options.

  • Web Browser. The web browser that the IDE uses for displaying web content. The following options are available in the drop-down list:
    • <Default System Browser>. Your computer's default web browser.
  • Proxy settings. Specify the general proxy settings the IDE uses to access the Internet by choosing one of the following.
    • No Proxy
    • Use System Proxy Settings
    • Manual Proxy Settings

      If you select this option, you must specify the HTTP Proxy and Proxy Port. The HTTP Proxy is the host name or numeric IP address of the proxy server to use to connect to HTTP and FTP sites. The Proxy Port is the port number of the proxy server. Click More to configure more advanced proxy settings.

  • Usage Statistics. Determines whether anonymous usage data will be sent to the NetBeans team to improve the IDE.

Note.

  • Maven will ignore proxy settings set in this panel. For details about Maven proxy settings, see the following FAQ.
