Add Ability

Use to associate an ability with a project configuration or to create new abilities.

Element Description

Ability Name

Lists abilities that have been created previously in the IDE.

Select one from the drop-down list or enter a name.

Value

(Optional) Specifies the value of the selected ability. For example, a value can be an integer, string, or empty symbol used as a boolean.
