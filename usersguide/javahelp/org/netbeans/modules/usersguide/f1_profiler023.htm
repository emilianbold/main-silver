New Profiling Point Wizard: Timed Take Snapshot

Use to take profiling snapshots or heap dumps at a certain time or after a certain interval. ?

Element Description

Name

Specifies the name that you want to use to identify the profiling point.

This name is used to identify the profiling point in the Profiling Points window.

Settings

Snapshot options

Take

Specifies the type of snapshot that you want to take.

Save

Specifies the location for the saved snapshots. You can save snapshots to the default directory in your project or you can specify a location on your local system.

Reset results after taking snapshot

Resets the collected profiling results after each snapshot is taken.

Time

Specifies when you want the IDE to take the snapshot.

If you select Take Once, you need to specify a time and the IDE takes a single snapshot at the specified time.

If you select Take Every, you need to specify the interval between snapshots. The IDE takes snapshots at the specified interval while the profiling session is running.
