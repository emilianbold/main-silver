Add or Duplicate Project Configuration Dialog

Use to add new project configurations to the active selected project.

Element Description

New Configuration Name

Specifies the name of the new configuration.

Alternatively, you can choose a project template from the Use Configuration Template drop-down menu. The new configuration contains all the properties of the configuration template.

You can customize the properties for the new configuration by choosing property categories in the right pane of the Properties dialog.
