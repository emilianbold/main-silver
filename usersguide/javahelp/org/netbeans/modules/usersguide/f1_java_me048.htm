New Mobile Application Wizard: More Configurations Selection Page

Use to choose additional platform configurations for the project. The configurations are then immediately available to you for testing and compiling your project.

Note: After you create the project, you can create additional project configurations with alternate platform settings.

Element Description

Project Configuration Templates

Displays the list of configurations from all the configuration templates that have been created in the IDE.

Select the configurations you want to add.
