Local History: Revert To Dialog Box

The Local History Revert To dialog box enables you to revert a current version of a file to a previous version, as maintained in the IDE's local history repository.

You access this dialog box by right-clicking a file or folder in the Projects or Files window and choosing Local History > Revert to.

The dialog box is displays a two-column table presenting all previous versions of the selected file, listed in chronological order. You can revert to any version by selecting the corresponding table row and clicking Revert.

Element Description

Version

Versions are arranged by date and named accordingly.

Label

Labels can be added to specific versions. To add a label to a version, click in the label text area for the corresponding version and type in a label.
