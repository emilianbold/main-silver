Add Server Instance Wizard: JBoss Server Instance Properties

You see this page of the Add Server Instance wizard when you are registering a JBoss Application Server instance. Use this page to specify the domain you are registering.

Select the domain from the Domain drop-down menu.

If you want to change a domain's port number, you must do so in the domain's server.xml file. By default, the port number is 8080. For example, if you are using the default domain, the domain's server.xml file is found here within the JBoss installation directory:

\server\default\deploy\jbossweb.sar\server.xml
