Options Window: Miscellaneous: SVG

Use to configure the IDE to launch an external program to edit Scalable Vector Graphics (SVG) files.

Choose Tools > Options, select the Miscellaneous option, and click the SVG tab to open this dialog box.

Element Description

Snippets

Specifies the registered custom snippet to use.

External Editor Path

Specifies the external SVG editor to use.

Click Browse to navigate to the editor's location.

Export

Exports the selected SVG options to the specified ZIP file.

Import

Imports the selected SVG options to import from the specified source.
