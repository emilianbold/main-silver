Debugging: Instances Window

Use to displays the instances and referers for the class that you select in the Loaded Classes window. Right-click a class in the Loaded Classes window and choosing Show in Instances view to open this window. The Instances view requires that the IDE and your application are running on JDK 1.6.The Instances window is composed of the three panes.

Element Description

Instances Browser

Displays instances for the selected class.

Select an instance to browse the fields of the instance in the the Fields browser.

Select an instance to browse the references to the instance in the References browser.

Fields Browser

Displays fields for the selected class.

Right-click in this browser to view an instance, an object, or Java source

Show Instance

Shows the instance in the Instances browser.

Show in Loaded Classes View

Displays the object in the Loaded Classes window.

Go to Source

Opens the Java source file in the Source Editor.

References Browser

Displays referers for the selected class.
