Import Mobile Project from Existing MIDP Sources Wizard: Specify MIDP Sources page

Use to specify the locations of the moblie project's source files and the Java Application Descriptor file.

Element Description

Sources Location

Specifies the root folder (the folder above the first package) that contains the Java source code for the project.

JAD/Manifest Location

Specifies the file that contains the Java Application Descriptor (JAD) for the project. JAD files are indicated by the extension .jad.
