MIDP Project Properties: Abilities Page

Use to create new abilities and associate them with project configurations.

Element Description

Add

Opens the Add Ability dialog box with which you associate an ability with the current configuration.

Edit

Opens the Edit Ability dialog box with which you can edit the selected ability.

Remove

Removes the selected ability.
