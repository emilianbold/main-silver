Add/Edit Permission for API

Use to add permissions to be requested by the MIDlet Suite. The attributes are listed in the in the Java Application Descriptor (JAD) file.

Element Description

Permission

Lists available API permissions.

Protected Resource Name

Displays the name of a protected resource.

Select the Use Null option to create no name for the protected resource.

Actions Requested

Specifies actions which are requested for the selected Permission.
