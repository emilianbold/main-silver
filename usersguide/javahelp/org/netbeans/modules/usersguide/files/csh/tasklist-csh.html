Action Items Window

See Also

The Action Items window displays a unified list of things that you need to resolve in your various files and projects. You can view the following types of entries in the list of action items.

  • Compiler errors
  • Hudson tasks
  • Issues
  • Maven POM problems
  • TODO
  • White list violations

You can use filters to determine the entries that are displayed in the list.

To open the Action Items window, choose Window > Action Items (Ctrl-6) from the main menu. You can sort the list by clicking the column heading of the Action Items window. The Action Items toolbar enables you to modify and work with the displayed entries.

You can do the following in the Action Items window:

  • Jump to an action item entry or issue.
  • Sort and filter the entries displayed in the window.
  • Create and modify filters for action items.

Action Items Window Toolbar

The Action Items window toolbar is displayed on the left side of the window and has the following buttons.

  • Current File. When selected, the IDE scans the file that is currently selected in the editor and displays a list of the action items found in that file. When you open a new file in the editor, the file is scanned and only the action items in that file are displayed.
  • Selected projects. When selected, the IDE scans all the files in the currently selected project and displays all the action items in the project in the window.
  • All opened projects. When selected, the IDE scans all the files in all open projects and displays a list of all the action items in the window.
  • Filter. Click the Filter button to modify the filters for the action items or to create a new filter. You can click the drop-down list on this button to select a filter to apply to the list of action items.
  • Group Action Items By Category. When selected, the list of action items is grouped by category.

Issues Marked as Action Items

The IDE provides support for some issue trackers that enables you to work with issues from within the IDE. If you are using a supported issue tracker, you can add an issue to your personal list of action items by clicking Add to Action Items in the tab for the issue. You can click an issue entry in the Action Items window to open the issue tab in the main window. You can right-click the issue entry to open a popup menu that enables you to remove the issue from the list.

For details about adding an issue to your list of action items, see Working With Issues.
