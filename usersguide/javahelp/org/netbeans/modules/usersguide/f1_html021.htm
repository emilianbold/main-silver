Options Window: HTML/JS: Gulp

You use this panel in the Options window to configure the IDE to use Gulp.

This panel contains the following elements.

Element Description

Gulp Path

 Specify the path to the Gulp cmd file.
    Install Gulp Click to display and install Gulp on your machine.
