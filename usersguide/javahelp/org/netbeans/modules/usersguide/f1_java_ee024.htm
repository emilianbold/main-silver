Add Library Dialog Box (EJB Module)

You open this dialog box by right-clicking the Libraries node of an EJB module and choosing Add Library. The Add Library dialog box lets you choose the libraries that the module or application contains on its classpath. The Add Library dialog box lists the libraries registered with the Ant Library Manager.

To add a library to the list of available libraries, click Manage Libraries to open the Ant Library Manager. Alternatively, choose Tools > Ant Libraries from the main menu to open the Ant Library Manager.
