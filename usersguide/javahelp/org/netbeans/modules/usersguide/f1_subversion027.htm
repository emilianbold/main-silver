Subversion: Browse Repository Folders Dialog Box

The Subversion Browse Repository Folders dialog box enables you to select a folder when checking out from the repository.

The dialog box displays the contents of the repository in a tree view. Expand the nodes to view and select folders in the repository. To specify a previous revision, click the ellipsis button for a given folder under the History column.

When you click OK, the path to the selected module appears directly in the field of the wizard or dialog you are working from.

Note: You can select more than one location by holding down the Ctrl key when selecting folders. However, multiple selection works only for files and folders at the same level.
