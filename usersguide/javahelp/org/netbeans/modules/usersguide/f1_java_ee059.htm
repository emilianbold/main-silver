Service Reference Port Information: Stub Property Editor

Use to add or edit stub property values for the service reference port. How?

The following stub values and properties can be set in the property editor:

Element Description

USERNAME_PROPERTY

Sets the value to the user name.

PASSWORD_PROPERTY

Sets the value to the user's password.

SESSION_MAINTAIN_PROPERTY

Sets the value to True or False.

ENDPOINT_ADDRESS_PROPERTY

Sets the value to the address of the service port endpoint.
