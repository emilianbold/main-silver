New RESTful JavaScript Client Wizard: Configure JavaScript REST Client

In this panel you specify the REST resource classes.

This panel contains the following fields, which specify the JavaScript file name and the REST resource.

Element Description

File Name

Name of the REST JavaScript client file. The default name is RestClient.

Project

The NetBeans IDE project containing this web client. The value is generated by the IDE.

Folder

The parent directory inside the NetBeans IDE project that will contain the JavaScript file.

REST Project Resource

Click Browse to locate the REST resource in the project.

Choose resulting UI

(Optional) Select a UI in the dropdown list.

Add Backbone.js to project sources

(Optional) When selected the IDE automatically adds the Backbone.js JavaScript library to the project.
