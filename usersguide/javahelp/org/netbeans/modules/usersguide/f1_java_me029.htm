MIDP Project Properties: Application Descriptor

Use to view and modify the attributes of the Java Application Descriptor (JAD) file of a project configuration.

Element Description

Project Configuraton

Specifies the active project configuration is listed.

Open the drop-down list to select another configuration.

Click Manage Configuration to add, modify, duplicate, or remove a project configuration.

Attributes

Lists the required, optional, and user-customized attributes of the selected project configuration.

MIDlets

Lists the MIDlets contained in the MIDlet suite for the selected project configuration.

Push Registry

Lists Push Registry Entries for classes in the selected projected configuration.

API Permissions

Lists the API Permissions assigned for the selected project configuration.
