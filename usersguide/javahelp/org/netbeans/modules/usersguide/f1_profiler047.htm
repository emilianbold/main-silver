Add Filter

Use to add and define a new filter.

The dialog contains the following elements for defining a new filter.

Element Description
Name Specifies the name of a new filter.
Color When selected, the custom color for the matching results can be defined by clicking the Default color button. When deselected, the matching results are displayed using the default color.
Value Specifies the value of a new filter.
