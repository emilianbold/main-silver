MIDP Project Properties: Creating JAR

Use to modify JAR and JAD creation settings in a MIDP project.

Choose Build > Creating JAR from the project properties dialog box to access this page.

Element Description

Use Values from "Default Configuration"

Enables you to customize the properties for a project configuration other than the default configuration if this option is unchecked.

JAD File Name

Specifies the name of the JAD file created by the project sources.

JAR File Name

Specifies the name of the JAR file created by the project sources

Compress JAR

Compresses the JAR file.
