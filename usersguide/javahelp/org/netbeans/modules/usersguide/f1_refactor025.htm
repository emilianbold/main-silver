Use Supertype Where Possible Dialog Box

Use to specify options for the Use Supertype Where Possible refactoring command. Use this command to change code that references the selected class (or other type) so that it references a supertype of the selected type.

Element Description

Supertype to Use

Displays Contains the name of the class to be moved.

From the drop-down list, choose the supertype that you want the code to be changed to reference.
