New JSF Pages From Entity Classes Wizard: JSF Configuration

This panel of the New JSF Pages From Entity Classes wizard enables you to modify the JavaServer Faces configuration. This panel is available when you add support for JSF to the application using the wizard. The panel contains three tabs for configuring JSF options.

Element Description

Libraries

The Libraries tab enables you to set the following properties to specify the version and location of the JSF libraries that are used in the project.

Server Library

When selected, you can choose from the list of JSF libraries that are registered with the server.

Registered Libraries

When selected, you can specify which registered library to use from the drop down list.

Create New Library

When selected you can create a new library by specifying the location of the JSF library.

JSF Folder

Type or browse for the location of the directory that contains the JSF library.

Library Name

Type the name of the JSF library.

Configuration

The Configuration tab enables you to set the following properties.

JSF Servlet URL Mapping

Specify the servet URL mapping. The default setting is /faces/*.

Preferred Page Language

Select from the available page languages in the dropdown list.

Components

The Components tab enables you to add JSF component suites to the application. If you select a component suite you might need to explicitly create and register a library for the component suite and specify the location of the required libraries.
