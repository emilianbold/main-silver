Options Window: Keymap

You use this panel in the Options window to set the keyboard shortcuts for the IDE. You can change between pre-configured shortcut sets, create a new shortcut set, or change the keyboard shortcut for an individual command.

You can access this panel by choosing Tools > Options from the main menu, and then clicking the Keymap category.

Element Description

Profile

Select a profile from the list to switch between profiles.

Search

Type in the text field to filter the shortcuts by the name of the command.

Search in Shortcuts

Type in the text field to filter the list to the shortcuts containing the keyboard character or characters.
