Versioning Options: Local History

The options in this dialog affect the Local History view of files in the Editor.

  • Clean up Local History files after x days. Set the number of days between cleanups of local history files. You must enter a whole number. The default is 7.
    • Clean up only versions without a label. Check this box and the IDE will not delete Local History versions that you labeled. To label a version of a file, open the Local History view of the file, select the version, and type something in the label field.
    • Keep Local History files forever. Check this box and the IDE will never delete file versions in local history. Note that this can add up to a lot of disk space over time.
