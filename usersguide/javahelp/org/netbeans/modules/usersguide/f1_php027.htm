Zend Framework Options

Element Description

Zend script

The path to the Zend Framework run script and default parameters.

Search

The IDE searches for all Zend Framework run scripts that you have installed. A dialog opens in which you select which run script to use.

Browse...

Manually browse your file system for the run script.

Register Provider

If you are using Zend Framework 1.10 or newer, register the provider. Look at the output window after you click Register Provider. The IDE displays the result of registering the provider in the Output windows. If registration failed, you may have a problem in your zf.php or Zend Framework run script.
