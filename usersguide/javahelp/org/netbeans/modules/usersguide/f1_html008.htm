HTML5 Project Properties Dialog Box: JavaScript Files

You open the HTML5 Project Properties dialog box by right-clicking any HTML5 project node in the Projects window and choosing Properties.

The JavaScript Files page enables you to add JavaScript libraries to your project. To add the JavaScript file, select the file that you want to add from the list of available files and click the right-arrow button (>) to add the file to the list of selected files.

You can filter the list of available files by typing in the text field above the list of available files.

You can select the version of the JavaScript file that you want to add to the project by clicking the version in the Version column.

You cannot use this dialog box to remove the JavaScript files from the project.
