Options Window: Miscellaneous: Windows

You use this panel to customize the NetBeans window system's functionality.

You can set the following options:

Element Description

Drag window image

If selected, the IDE displays a small copy of the window content when dragging a window.

Transparent drag window image

If selected, the IDE displays a small transparent copy of the window content when dragging a window.

Snapping floating windows

If selected, undocked windows snap to edges of other undocked windows when being dragged.

Snap to screen edges

If selected, the IDE snaps an undocked window to the edges of the screen.

Transparent floating windows

If selected, the IDE makes floating windows semi-transparent a few seconds after they go out of focus.

Document tabs

Modify the position of tabs in the document area by using the following options:

    New document opens next to active document tab

If selected, a new document opens in the same docking spot next to the active document tab.

    Close activates most recent document

If selected, the most recent document becomes active after an active document is closed.

    Tabs placement

Select where to place the editor tabs. You can choose top, bottom, left or right.

    Multi-row tabs

If selected, the editor tabs display in multiple rows.

Native Look and Feel

The IDE uses platform-specific user interface components if the components offer sufficient functionality.

    Maximize use of native look and feel

If selected, the IDE uses native user interface components (for example, native file chooser on OS X).
