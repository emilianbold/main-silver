Issue Tracking: Issue Tracker Server Response Dialog Box

This dialog opens when the IDE encounters a problem when attempting to contact the issue tracking server and displays the response that the IDE received from the server. The dialog provides information that might help you resolve the problem.

In some cases the dialog may display links to a web page for additional details. It is also possible that partially rendered HTML may be displayed in the dialog.
