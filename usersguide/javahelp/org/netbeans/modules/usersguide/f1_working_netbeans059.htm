Project Group: Create New Group Dialog Box

The Create New Group dialog box enables you to create groupings of projects so that you can open and close several projects at once. In addition, each group can have a different main project.

You can open the Create New Group dialog box by choosing File > Project Group > New Group.

You can use the dialog box to create the following types of groups:

Element Description

Free Group

When you create a free group, you can choose either of the following options:

    Use Currently Open Projects

Adds the currently open projects to the group.

    Automatically Save Project List

Changes the contents of the project group according to the projects that you open and close when the project group is open.

Deselect this option if you do not want the contents of a free group to change as you open and close projects.

Project and All Required Projects

Creates a group that consists of a master project and all of the projects that it depends on.

Folder of Projects.

Creates a group out of all of the projects that are within a folder you specify.
