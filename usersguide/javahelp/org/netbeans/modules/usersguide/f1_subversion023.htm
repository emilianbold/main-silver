Subversion: Search Revisions Dialog Box

The Subversion Search Revisions dialog box enables you to specify a revision for the repository folder you previously select when examining data from the repository.

You can search for a revision by entering a date in the log entries field (YYYY-MM-DD format) to set a beginning point for the search. Leave this field empty to select all available revisions.

Click the List button to initiate the search. All revisions based on the search criteria display in the text area.

When you select a displayed revision from the text area and click OK, the selected revision appears in the Repository Revision field of the wizard or dialog you are working from.
