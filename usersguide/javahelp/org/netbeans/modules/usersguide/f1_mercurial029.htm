Mercurial: Clone External Repository Wizard: Destination Directory

The Mercurial Clone External Repository wizard enables you to create a local clone of a Mercurial repository.

This panel contains the following elements:

Element Description

Parent Directory

Specify the parent directory for the clone. Click Browse to open an explorer to navigate to a location on your computer.

Clone Name

Specify a name for the clone.

Scan for NetBeans Projects after Clone

.When selected, the IDE scans for any NetBeans projects in the clone and will prompt you if you want to open them.
