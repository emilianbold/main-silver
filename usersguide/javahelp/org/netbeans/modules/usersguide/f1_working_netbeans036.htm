Tasks: Select Keyword Dialog Box

You use this dialog box when you want to assign a registered keyword to a task or search for tasks that contain the selected keyword.

This dialog displays a list of possible keywords that can be assigned to the task.

Element Description

Keywords

 Select the keyword or keywords that you want to add to the task or query. The list of keywords is determined by the administrator of the project's task repository.
