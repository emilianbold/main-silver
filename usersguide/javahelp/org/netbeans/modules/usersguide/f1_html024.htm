Options Window: HTML/JS: Nashorn

You use this panel in the Options window to configure the IDE to use Oracle Nashorn JavaScript engine.

This panel contains the following elements.

Element Description

Java Platform

 Select a JDK with Oracle Nashorn to be used or click Manage Platforms to register a version of JDK that contains Oracle Nashorn.
Engine Options Specify options to customize Oracle Nashorn script engine.
Arguments Specify arguments for jjs invocation.
