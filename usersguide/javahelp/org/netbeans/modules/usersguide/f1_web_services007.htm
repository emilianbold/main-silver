Standard Web Project Properties Dialog Box: Documenting

You open the Standard Web Project Properties dialog box by right-clicking any and choosing Properties.

The Documenting page lets you configure how the IDE generates Javadoc documentation for your classes.

You can set the most commonly used Javadoc options by selecting the checkboxes in the dialog box. You can also enter additional Javadoc options in a space-separated list in the Additional Javadoc Options field.
