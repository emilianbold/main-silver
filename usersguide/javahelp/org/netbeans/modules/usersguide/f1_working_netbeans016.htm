Options Window: Miscellaneous: Issue Tracking

You use this panel to configure general properties for how the IDE interacts with issue trackers. You can access this panel by choosing Tools > Options from the main menu, clicking the Miscellaneous category and then clicking the Issue Tracking tab.

The default issue tracking system is Bugzilla.

You can add issue trackers in the Services window.

Element Description

Issue Tracking Systems

Select an issue tracking system from the list to modify the properties of that tracker.

Refresh opened issues (Bugzilla)

Specify how often the IDE checks open issues for changes to the issue.

Refresh opened queries

Specify how often the IDE checks open queries for udates.
