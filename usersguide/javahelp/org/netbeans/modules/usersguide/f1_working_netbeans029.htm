Issue Tracking: Save Issue Query Dialog Box

Use this dialog to assign a name to a saved issue tracking query. The saved query appears in the Find Issues tab. You can click the saved query at any time to search issues using the saved criteria.

Element Description

Name

Type a name to use to identify the query.
