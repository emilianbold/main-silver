Web Service Attributes Editor: WSDL Customization: Services

Use the Services section to customize the name of the Java class that will be generated for the Service interface.

For each service you can set the following property.

Element Description
Java Class If Use Default is deselected you can type the name of the Java class to be generated. If Use Default is selected wsimport defaults to the value of the name attribute of the wsdl:service element.
