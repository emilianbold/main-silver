Build Selected Project Configurations

Use to choose which project configuration to build for your project. The IDE then builds a distribution JAR for each configuration you select.

Element Description

Selected Project Configurations

Lists the project configurations.

Check the configurations you want to include in the project's batch build.

Uncheck the configurations you want excluded from the build.
