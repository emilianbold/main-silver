Web Browser Window

The Web Browser window displays an embedded web browser that is based on the WebKit browser engine. You can use the embedded WebKit browser to browse web pages and as the target browser for web applications.

You can set Embedded WebKit Browser as the default browser for displaying web pages in the Web Browsers dropdown list in the General category in the Options window.

The Web Browser window contains the following elements.

Element Description

Location Bar

Displays the location of the current page.

Toolbar

(HTML5 applications only) The toolbar contains elements that enable you to change the display screen size.
