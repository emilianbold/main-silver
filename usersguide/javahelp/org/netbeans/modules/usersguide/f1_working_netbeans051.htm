Select The Diff Patch Dialog Box

The Select The Diff Patch dialog box enables you to apply patches to code. Patched binary files are not displayed.

You access this dialog box by selecting a file or folder in the Projects or Files window and either:

  • choosing Tools > Apply Diff Patch from the main menu.
  • right-clicking a file or folder and choosing Tools > Apply Diff Patch.

The Select The Diff Patch dialog box contains a file browser that you use to locate and select the file and the following option for selecting file types:

Element Description

File Format

Select the file formats that are displayed in the dialog box.
