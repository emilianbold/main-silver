Login to netbeans.org Dialog Box

This dialog box opens when you choose to submit an issue with NetBeans IDE and you are not logged in to netbeans.org.

You can report a problem with NetBeans IDE and submit an issue to the netbeans.org issue tracker by choosing Help > Report Issue from the main menu. To report an issue from the IDE you need to have an account at netbeans.org and you must be logged in. After you log in, you can submit an issue in the New Issue window.

If you do not have an account, you can click the link in the dialog box to open netbeans.org in your browser and create an account.
