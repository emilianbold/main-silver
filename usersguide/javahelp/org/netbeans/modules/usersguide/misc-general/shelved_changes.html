Shelved Changes Dialog Box

See Also

The Shelved Changes dialog box enables you to get the temporarily shelved changes back into a working directory.

You access this dialog box by choosing Team > Shelve > Unshelve Changes from the main menu.

To define the patch to be unshelved, you need to specify the following options:

  • Patches. Choose the name of a patch you want to unshelve.
  • Keep Patch File. When selected, the shelved patch file is not removed from the patch storage folder.

When you click Unshelve Changes, the IDE gets the changes shelved in a patch file back into the working directory.

When you click Remove Patch, the IDE deletes the selected patch file with the shelved changes from the storage folder.
