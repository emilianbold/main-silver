Enterprise Application Client Project Properties Dialog Box: Compiling

Use the Compiling panel to set general compilation options for your project. To open the Enterprise Application Client Project Properties dialog box, right-click any application client project node and choose Properties. Under the Build tree, choose Compiling.

For more infomation on compiler options, see:

To add any additional compiler options, enter them in a space-separated list in the Additional Compiler Options text field.

Notes:

  • Do not use compiler options like -bootclasspath to cross-compile between Java platforms. Instead, register the target Java platform and set the target JDK for the project.
  • You can manage the project's classpath by clicking Libraries in the left panel of the dialog box. To set up file filters for JAR file creation, click Packaging.
