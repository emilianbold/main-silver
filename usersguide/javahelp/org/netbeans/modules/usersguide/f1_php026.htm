Symfony2 Options

This table lets you control the following options for Symfony2:

Element Description

Symfony

The path to the Symfony Standard Edition .zip archive. If you do not have a copy of Symfony Standard Edition, click the link.

Browse...

Manually browse your file system for the Symfony2 .zip archive.

Ignore cache directory by default

Deselect this for NetBeans IDE to include the /cache directory when scanning for Symfony functions. This can be useful if you cache actions.

This box globally sets whether cache directories are ignored. You can set whether the cache directory is ignored for an individual project in that project's Properties, under Frameworks > Symfony.
