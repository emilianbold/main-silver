Web Browser Window

See Also

The Web Browser window displays an embedded web browser that is based on the WebKit browser engine. You can use the embedded WebKit browser to browse web pages and as the target browser for web applications.

You can set Embedded WebKit Browser as the default browser for displaying web pages in the Web Browsers dropdown list in the General category in the Options window. When set as the default browser, you can open the Web Browser window by choosing Window > Web > Web Browser from the main menu. When set as the default browser, the Web Browser window opens automatically when you run a web application.

The Web Browser window contains the following elements.

  • Location Bar. Displays the location of the current page.
  • Toolbar. (HTML5 applications only) A toolbar contains the following elements that enable you to change the display screen size.
