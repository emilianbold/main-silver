Web Panel

The Web panel is only provided for web application projects. The Web panel displays the name of the servlet included in the web application. If the servlet is locked, the panel displays an Unlock button that enables you to edit the properties of the servlet.

Properties of the servlet that can be edited in the Web panel include:

  • Web Application Display name

  • Enable (the default setting) or disable inclusion of the servlet

  • Servlet Name

  • Servlet Mapping

  • Enable (the default setting) or disable Browse This Servlet on Run
