Remove a Device From a Platform

A device can be removed from a platform by using either the Java Platform Manager dialog or by using the Services window and shortcut menu.

Use Java Platform Manager to Remove a Device

  1. Choose Tools > Java Platforms from the main window.
  2. Select Java Card platform.
  3. Click Manage Devices.
  4. Select the device name and click Remove.
  5. Click OK to close the Device manager.
  6. Click Close to close the Java Platform Manager.

Use Services Window to Remove a Device

  • Right click the device name and select Delete.
