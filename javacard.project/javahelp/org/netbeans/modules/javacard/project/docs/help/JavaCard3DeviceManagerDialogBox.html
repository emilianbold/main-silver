Java Card 3 Device Manager Dialog Box

The device manager dialog box enables you to add or remove Java Card devices and to set their properties. Properties that can be set include:

  • RAM Size - default value is 1M

  • EEPROM Size - default value is 8M

  • COR Size - default value is 4K

  • Logger Level - default setting is Debug

  • HTTP Port - default value is 8019

  • Contacted Port - default value is 9025

  • Contacted Protocol - default value is Block-Oriented (T=1)

  • Contactless Port - default value is 9026

  • Debug Proxy <-> CJCRE Port - default value is 7019

  • IDE <-> Debug Proxy Port - default value is 7020

  • Suspend Threads on Startup - default value is unselected

  • Server URL - default value is http://localhost:8019

  • Card Manager URL - default value is http://localhost:8019/cardmanager

  • Card Manager is on a remote computer - default value is unselected
