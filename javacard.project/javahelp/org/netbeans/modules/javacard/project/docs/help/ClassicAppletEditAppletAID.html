Edit an AID of a Classic Applet

  1. Edit the RID value.

    The RID is a 5 byte hexadecimal value. The RID value is the unique ID for your organization, assigned by a standards body. A default RID is provided by the plugin and can be used during development. However, before deployment you must obtain a valid RID from the ISO.

  2. Edit the PIX value or click Generate to create a new PIX value.

    The PIX (proprietary identifier) value uniquely identifies an individual applet. It is a 0 - 11 byte hexadecimal value generated by the IDE or manually set by the developer.
