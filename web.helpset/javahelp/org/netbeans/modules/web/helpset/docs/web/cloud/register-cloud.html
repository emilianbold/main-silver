Registering a Cloud Account in NetBeans IDE

See Also 

Register cloud accounts in the Services window (Ctrl-5).

  1. Right-click the Cloud node in the Services window and select Add Cloud. A dialog opens listing supported cloud providers.
  2. Select the Cloud provider and click Next. A panel opens with options specific to the selected cloud provider.
  3. Complete the dialog with the details of your cloud account. Click Next.
  4. The IDE attempts to communicate with your cloud account. If this communication is successful, a list appears of the resources associated with your cloud account. Click Finish.

After registering a cloud account, the web/application servers associated with the account appear in the Services window under the Servers node.

Amazon Beanstalk Service

Sign up for Amazon Elastic Beanstalk Service from your Amazon AWS Management Console. NetBeans IDE only requires your key and secret key. Note that NetBeans only supports the Amazon Beanstalk Tomcat servlet containers, not Apache PHP web servers.
