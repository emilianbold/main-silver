Cloud Providers

See Also 

In the Cloud Providers dialog, you can add a cloud provider, remove a cloud provider, or change some details of a cloud provider. To open the Cloud Providers dialog, go to Tools > Cloud Providers.

Add Cloud/Remove Cloud

To add a cloud provider, select Add Cloud. To remove a cloud provider, select a provider from the list and click Remove Cloud.

Change Amazon Elastic Beanstalk Properties

If you select an Amazon Elastic Beanstalk provider from the list, you can change the public key and secret key.
