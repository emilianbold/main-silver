Standard Web Project Properties Dialog Box: Frameworks

See Also

You open the Standard Web Project Properties dialog box by right-clicking any  and choosing Properties. In the Frameworks page, you choose one or more frameworks that you want to add to your web application.

Click the Add button and then choose one or both of the following:

  • JavaServer Faces: For details, see About JavaServer Faces Framework Support.
  • Spring Web MVC: For details, see About Spring Web MVC Framework Support.
  • Struts 1.3.8: For details, see About Struts Framework Support.
  • Hibernate 3.2.5: Specifies that you want the IDE to add Hibernate 3.2.5 libraries to the project classpath and create a Hibernate configuration file (hibernate.cfg.xml) for your application.
