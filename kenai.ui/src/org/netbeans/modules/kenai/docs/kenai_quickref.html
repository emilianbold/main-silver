Team Project Tasks: Quick Reference

The following table describes common Team project tasks.

To perform this task Follow these steps
Register a Team Server
  1. In the Services window, right-click the Team Servers node and choose Add Team Server.
  2. Type the name and URL of the Team Server. Click OK.
Log in to a Team Server
  1. From the main menu, choose Team > Team Server > Login.

    You need to register a Team Server and have an account before you can login.
Add a local project to a Team Server
  1. Right-click the project in the Projects window and choose Share on Team Server.
  2. Specify the details for the new Team project in the wizard. Click Finish.
Create a new Team project
  1. From the main menu, choose Team > Team Server > New Team Project.
  2. In the New Team Project wizard, specify the project name, title and license. Click Next.
  3. Specify the source code repository and issue tracking details.
  4. Review the details of the new project. Click Finish.
Open a Team project
  1. From the main menu, choose Team > Team Server > Open Team Project.
  2. Type the search terms. Click Search.
  3. Select the project in the list. Click OK.

    When you open a Team project, the project is listed under Open Projects in the Team Dashboard.
View projects that you joined
  • Choose Window > Team from the main menu to open the Team Dashboard.

    Projects that you joined are listed under My Projects.
View open projects in the Team Dashboard
  • Choose Window > Team from the main menu to open the Team Dashboard.

    Team projects that are open are listed under Open Projects. When a Team project is open in the dashboard, you can see an overview of the details of the project.

Retrieve sources of a project on a Team Server
  1. Do one of the following:
    • Expand the Sources node of the project in the Team Dashboard and click "get".
    • Choose Team > Team Server > Get Sources from the main menu.
  2. Select the repository for the project.
  3. Specify the location for the local repository. Click Get From Team Server.
Log in to chat
  1. Choose Team > Team Server > Login from the main menu.
  2. Enter your username and password in the Login to Team Server dialog box.
  3. Select Online on Chat. Click Login.

    If you are already logged in to a Team Server, click the Offline icon (  ) in the status bar.
Enter a chat room or initiate a private chat
  1. Log in to Chat.
  2. Choose Window > Chat in the main menu to open the Chat window.
  3. Double-click the project name or member name.
To delete a Team project
  • Log in to the Team Server site in your web browser.
  • Open the Settings tab of the Manage page of the project.
  • Click Delete.

    Only the project owner can delete a Team project.
