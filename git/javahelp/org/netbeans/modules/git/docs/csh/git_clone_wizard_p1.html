Git Clone Repository Wizard: Repository

See Also

You use the Clone Repository wizard to get a copy of an already existing Git repository.

To open the Clone Repository wizard, choose Team > Git > Clone.

At the Repository page, you have to set the following properties:

  • Repository URL. Select a Git root protocol and location from the drop-down list. The drop-down list contains a list of supported protocols, as well as previous Git root locations used in the IDE.
  • User. Enter your user name for access to the repository. (Required for http and https protocols.)
  • Password. Enter your password for access to the repository. (Required for http and https protocols.)
  • Save Username and Password. Select the option if you want the IDE to remember your user name and password.
  • Proxy Configuration. Click the Proxy configuration button to specify details of your proxy configuration to connect to the Git repository.
