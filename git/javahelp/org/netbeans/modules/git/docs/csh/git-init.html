Git: Repository Root Path Dialog Box

See Also

The Git Repository Root Path dialog box enables you to initialize a Git repository either from existing files that are not in source control yet, or from an empty directory.

You can access the Repository Root Path dialog box by choosing either:

  • Versioning > Initialize Git Repository from the NetBeans project's right-click menu.
  • Team > Git > Initialize from the IDE's main menu.

In the Repository Root Path dialog box, specify the path to the repository you are going to store your versioned files in the Repository root path dialog box or click Browse and browse for the directory required.

The IDE initiates the Git initialize action. You can view files being added to the repository from the Output window (Ctrl-4).

The repository files are placed under a .git directory under the IDE project directory.
