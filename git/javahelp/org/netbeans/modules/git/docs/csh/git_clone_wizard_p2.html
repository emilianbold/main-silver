Git Clone Repository Wizard: Remote Branches

See Also

The Remote Branches panel of the Git Clone Repository wizard enables you to choose the branches to be included in the cloned repository.

You access this panel by arriving at the second panel of the Git Clone Repository wizard (Team > Git > Clone from the main menu).

To download a branch to your local repository, select it in the Select Remote Branches list.
