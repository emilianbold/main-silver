Git Pull from Remote Repository Wizard: Remote Branches

See Also

The Remote Branches panel of the Git Pull from Remote Repository wizard enables you to choose the branches of the original remote repository to pull the changes from.

You access this panel by arriving at the second panel of the Git Pull from Remote Repository wizard (Team > Git > Remote > Pull from the main menu).

To choose a branch to pull changes from, select it in the Remote Branches list and click Finish.

Your local repository is synchronized with the origin repository.
