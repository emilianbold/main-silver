Git Fetch from Remote Repository Wizard: Remote Branches

See Also

The Remote Branches panel of the Git Fetch from Remote Repository wizard enables you to choose the branches of the original remote repository to get the changes that you do not have yet from.

You access this panel by arriving at the second panel of the Git Fetch from Remote Repository wizard (Team > Git > Remote > Fetch from the main menu).

To choose a branch to fetch changes from, select it in the Select Remote Branches list.
