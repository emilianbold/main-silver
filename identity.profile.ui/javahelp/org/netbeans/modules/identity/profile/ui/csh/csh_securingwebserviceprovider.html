Web Service Attributes Editor: Web Service Provider Security Configuration

See Also

The Security tab of the Web Service Attributes Editor lets you configure message-level security for web service providers.

You can configure the following settings:

  • Enable Message Level Security. Specifies whether message-level security is enabled.
  • Security Mechanism.
    • Request. Specifies the security mechanism for the requests.
    • The certificate settings at the domain level for the Liberty, X509, and SAML security mechanisms will apply for the server you select in this dialog box. See Configuring Security Mechanisms for Sun Java System Access Manager Instances for information about changing global settings.
  • Server. Specifies the Access Manager server instance.
