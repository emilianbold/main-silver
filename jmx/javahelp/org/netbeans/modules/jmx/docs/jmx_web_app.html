Accessing MBeans  from JSP

In this chapter we cover the steps needed to make a JSP access JMX management information.

Using the JMX Agent and MBean Wizards

The use of these wizards is similar to their use in a Java Application Project.
In the context of a web application, the JMX Agent is not a runnable class. When generating the Agent, unselect the Create Main Method.

Implementing Your MBeans

Do this the usual way.

Creating, Naming and Registering your MBeans Within the Agent

Do this the usual way (in the JMX Agent init method).

Displaying MBean Information in a JSP

Simply add the following line of code to make your JSP access  JMX MBean data:
<%--
    ...

    // Access the JMX Agent. By doing so, your MBeans will be automatically registered.
    javax.management.MBeanServer server = <MyJMXAgent>.getDefault().getMBeanServer();

    // Get an MBean Attribute.
    Object obj = server.getAttribute(new ObjectName("AN MBEAN NAME HERE", "YOUR ATTRIBUTE NAME TO DISPLAY HERE");

    // Then implement your MBean Attribute value presentation...

    ...
--%>
