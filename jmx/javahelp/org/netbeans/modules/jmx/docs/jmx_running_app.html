Running Your Project with JConsole

  1. Click on Run/Run Main Project with JConsole...
  2. If this is the first time you are running a management action, a dialog box pops up notifying you that the build file will be updated. Click Accept.
  3. Your process is compiled and run like any Java application. JConsole is displayed.
  4. Close JConsole to stop the execution.

Debugging Your Project with JConsole

  1. Click on Run/Debug Main Project with JConsole...
  2. If this is the first time you are running a management action, a dialog box pops up notifying you that the build file will be updated. Click Accept.
  3. Your process is compiled and run like any Java application.  Debugger is started. JConsole is displayed.
  4. Close JConsole to stop the session.
  5. Stop the debug Session.
