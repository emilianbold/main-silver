Writing a JMX Agent


This chapter demonstrates the use of the Agent Wizard. Part of the information contained here can be applied to any Java application.

Any application that registers MBeans within an MBeanServer is a JMX Agent.
You can locally access the platform's MBeanServer by calling java.lang.management.ManagementFactory.getPlatformMBeanServer()
. Once you get a reference to the MBeanServer you can then register your MBeans.

Using the JMX Agent Wizard to Generate a JMX Agent

  1. Select File/New File...
  2. Select the Management category. In File Types select JMX Agent
  3. Type in a Name and a Package.
  4. A main method and some sample code are generated. If the current project is a Java Application project, the agent class will be set as the project's main class.  You can disable these features by unchecking the appropriate checkboxes.
  5. Click Finish>
  6. The JMX Agent source code is loaded in the editor.
