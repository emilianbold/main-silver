Sending MBean Notifications

An MBean can send notifications in any part of its implementation (Attributes, Operations, non-exposed methods, etc.)
This chapter shows the  things to do to send a notification when an attribute is set.
The code extract is based on an MBean that has been updated with JMX Notification support (delegation to a notification broadcaster and generation of sequence number handling). This can be done using the JMX MBean Wizard or using the action to update an existing MBean (See the Updating Existing MBeans chapter).

Adding Code to Send Attribute Change Notifications

In the editor, select the set<MyAttribute> method name. After your attribute is set, add the following code:

//Send a notification when the attribute is set
Notification n = new Notification(AttributeChangeNotification.ATTRIBUTE_CHANGE, this, getNextSeqNumber());
broadcaster.sendNotification(n);
