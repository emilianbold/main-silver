Subversion: Manage Connection Settings Dialog Box

See Also

The Manage Connection Settings dialog box enables you to view all connections that have been checked out in the IDE and modify the current repository settings.

You access this dialog box by selecting a versioned file or folder and choosing Tools > Options > Miscellaneous > Versioning > Subversion and clicking Manage Connection Settings.

You can specify the following:

  • Repository URL. Select a connection that is stored in the IDE and edit the required settings, such as a username and password, if needed.

Click Remove from History to have the IDE delete the specified repository connection.
