Subversion: Update To Revision Dialog Box

See Also

The Subversion Update To Revision dialog box enables you to synchronize your local working copy with a selected revision from a repository.

You can access the Update To Revision dialog box by choosing Team > (Subversion >) Update > Update To from the main menu.

Specify a revision number in the Revision text field or click Search to display the Search Revisions dialog box and choose from a list of available revisions.

  • HEAD stands for the most current revision.

Click Update to complete synchronization of your local working copy with a selected revision from a repository.
