Subversion: Select Tree To Diff Dialog Box

See Also

The Select Tree To Diff dialog box enables you to set up a repository folder against which the diff will be performed.

You access this dialog box by selecting a versioned file or folder and choosing Tools > Diff > Diff To Branch/Tag.

You can specify the following:

  • Repository File. Specifies the path to the file to be examined. Click Browse to select a branch or tag you need to diff the file to.
  • Repository Revision. Displays the revision of the currently selected repository. Click Search to specify another revision for the repository.

Click Select to have the IDE compare the specified folders and display the differences.
