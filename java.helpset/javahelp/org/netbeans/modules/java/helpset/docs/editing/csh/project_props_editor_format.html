Project Properties Dialog Box: Formatting

See Also 

The Formatting panel enables you to override the global formatting options and specify the source code formatting options for a specific project. To open the Formatting panel for a project, right-click the project node in the Projects window and choose Properties and select the Formatting category.

By default, global formatting options are applied to all projects. To configure global formatting options for the IDE, open the Options window and select the Formatting tab in the Editor category. Alternatively, you can click the Edit global options button in this panel to open the Formatting tab in the Options window.

You can choose from the following options.

  • Use project specific options. Select this button to configure specific formatting options and apply them only to the current project. You can choose whether to load formatting options from other projects or use manually modified options from this panel.
  • Load from other projects. Click this button and choose a NetBeans IDE project from which to import formatting options.

For details about the settings, see the Editor Options.
