Palette Window

See Also

The Palette window displays a list containing all of the components available for adding to the IDE's various editors. For GUI forms, the items available in the Palette include containers and other visual components, such as buttons, labels, menus, and panels.

To open the Palette window, choose Window > Palette from the main menu.

The Palette window view can be customized to alter the way items are sorted, to show or hide item names, or display different sized icons of the listed items. To customize the Palette window view, right-click anywhere in the Palette and choose the appropriate menu item from the contextual menu.

  • Using the Palette Manager, you can add, remove, and organize Palette window components such as Swing components, AWT components, Layouts, and beans.
