GUI Builder Connection Wizard: Select Source Event

See Also

The Connection wizard enables you to establish a connection between two components triggered by an event within a form without having to write code manually.

You can launch the Connection wizard by clicking the Connection Mode button () in the GUI Builder toolbar and selecting a component that will generate the event and then a component that will receive it in the GUI Builder or Navigator window.

In the Select Source Event page of the Connection wizard, you specify the following properties:

  • Source Component. The source component of the triggering event is automatically set for you after it is selected in the GUI Builder or Navigator window.
  • Events. Select the event of a source component on which the trigger component will react by expanding the event type directory nodes.
  • Event Handler Method. Accept the default handler method name or type in a new name.

Click Next to specify an operation to be performed on the target component.
