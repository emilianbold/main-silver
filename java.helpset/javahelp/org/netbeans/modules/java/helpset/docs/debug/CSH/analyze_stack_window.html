Debugging: Analyze Stack Window

See Also

The Analyze Stack Window enables you to quickly jump from a stack trace element to a corresponding line in the Editor.

You open this dialog box by choosing Window > Debugging > Analyze Stack.

Click the Insert StackTrace from Clipboard button after you copy a stack trace in the Output tab of the IDE, to get the stack trace inserted into the Analyze Stack Window.

Note: When the Analyze Stack Window is opened, the IDE automatically presses the Insert StackTrace from Clipboard button and inserts a stack trace from your clipboard (if applicable).

Double-clicking a line in the Analyze Stack Window opens the source file (if it is closed) and moves the cursor to the line where an exception was thrown.
