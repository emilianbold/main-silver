Debugging Java GUIs

See Also

When debugging a Java GUI form, you can use the Visual Debugger to help you locate the form elements in your source code.

To use the Visual Debugger, you run the application in the debugger and take a snapshot of the UI that you want to debug. You can then select items in the snapshot and locate the corresponding line is the source code. After you locate the lines in the source code for the elements that you want to debug, you can add breakpoints.

To use the Visual Debugger to locate GUI-related source code:

  1. Run the application in the Debugger.
  2. Choose Debug > Take GUI Snapshot in the main menu.

    The GUI snapshot will open in a tab in the main window.

  3. Right-click an element in the GUI snapshot and choose a command from the popup menu.
