About OSGi Bundles

An OSGi bundle is a JAR file with OSGi-specific manifest entries. OSGi bundles can be installed into implementations of the OSGi framework, such as Felix and Equinox.

The IDE provides wizards for creating OSGi bundles in the following types of applications:

  • Maven-based standard Java SE applications
  • Maven-based web applications
  • Maven-based NetBeans Platform applications
  • Ant-based NetBeans Platform applications
