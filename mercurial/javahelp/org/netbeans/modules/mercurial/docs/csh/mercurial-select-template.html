Mercurial: Select Template Dialog Box

The Mercurial Select Template dialog box enables you to select a message template for a commit message.

You can access the Select Template dialog by choosing Team > (Mercurial >) Commit from the main menu and clicking the Load Template button in the Commit dialog box.

In the Select Template dialog box, you can select the Automatically fill into commit message option to have the specified template message inserted as a commit message in the Commit dialog box during the commit.

After you click Load File, the IDE displays the Select Template File dialog box where you can browse for a required file with the message template and load it into the Template area of the Select Template dialog box.

After you click Save as File, you can save the message template from the Template area of the Select Template dialog box into a file on your system.
