Mercurial: Repository Root Path Dialog Box

See Also

The Mercurial Repository Root Path dialog box enables you to initialize a Mercurial repository from an existing NetBeans project that is not in source control yet.

You can access the Repository Root Path dialog box by choosing either:

  • Versioning > Initialize Mercurial Project from the NetBeans project's right-click menu.
  • Team > Mercurial > Initialize Project from the IDE's main menu.

In the Repository Root Path dialog box, specify the path to the project you are going to import into Mercurial in the Repository root path dialog box or click Browse and browse for the directory required.

The IDE initiates the Mercurial initialize action. You can view files being added to the repository from the Output window (Ctrl+4).

The repository files are placed under a .hg directory under the IDE project directory.
