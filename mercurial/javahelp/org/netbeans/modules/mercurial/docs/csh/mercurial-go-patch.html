Mercurial: Go To Patch Dialog Box

See Also

The Go To Patch dialog box enables you to switch to a particular patch in a patch queue series.

You access this dialog box by selecting a versioned project, file, or folder and either:

  • choosing Team > Mercurial > Queues > Go To Patch from the main menu
  • right-clicking the selected item and choosing Mercurial > Queues > Go To Patch

To define the patch to be applied to the selected project, file or folder, you need to select the name of a patch in the following field:

  • Patches. A list of all patches available in a stack.
  • Names of applied patches display in bold.
  • Choose Pop all applied patches to remove the applied patches from the top of the stack and updates the working directory to undo the effects of the applied patches.

When you click Go, the IDE applies the changes contained in the selected patch to the chosen project, file, or folder.
