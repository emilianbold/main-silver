Mercurial: Finish Patches Dialog Box

See Also

The Finish Patches dialog box enables you to turn already applied patches in a patch queue series into a regular changeset.

You access this dialog box by selecting a versioned project, file, or folder and either:

  • choosing Team > Mercurial > Queues > Finish Patches from the main menu
  • right-clicking the selected item and choosing Mercurial > Queues > Finish Patches

To define the patch to be finished, you need to select the name of a patch in the following field:

  • Patches. A list of all applied patches available in a stack.

When you click Finish Patches, the IDE turns all applied patches up to the selected patch into regular changesets.
