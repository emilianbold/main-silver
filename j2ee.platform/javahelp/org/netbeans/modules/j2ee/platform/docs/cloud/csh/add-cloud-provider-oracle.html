Add Cloud Provider: Oracle Cloud

See Also 

In this dialog you enter the details of the Oracle Cloud account you want to use with NetBeans IDE. To open this dialog, go to the Services window, right-click the Cloud node, select Add Cloud, and select Oracle Cloud.

To register Oracle Cloud with the IDE, enter the following information:

  • Identity Domain. The individual or group identity of your Oracle Cloud account. If you do not have an Oracle Cloud account, click Open a New Account.
  • Java Service Name. The name of the Java Service you created when you set up your Oracle Cloud account.
  • Database Service Name. The name of the Database Service you created when you set up your Oracle Cloud account.
  • Administrator. Your identity as Oracle Cloud administrator (the email you gave when you set up your Oracle Cloud account).
  • Password. Your Oracle Cloud administrator password.
  • SDK. Path to your local copy of the Oracle Cloud SDK. Click Configure to browse for this file.
