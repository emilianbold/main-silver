Add Server Instance Wizard: Domain Admin Port Info

See Also

You see this page of the Add Server Instance wizard when you are registering a remote domain instance of the GlassFish Server v1, GlassFish Server v2.x or Sun Java System Application Server. Use this page of the wizard to specify the host and communication port for the remote domain.

To register an instance of GlassFish Server v1 or v2.x you need to install the GlassFish Server v1, v2.x plugin from the Update Center.

Enter the following information for the remote domain:

  • Host. Enter the host name or IP address for the remote GlassFish application server or Sun Java System Application Server.
  • Port. Enter the port number used to communicate with the remote server. The default port is 4848.

Click Next if you want to supply a username and password for the remote domain. Click Finish if you do not want to supply a username and password at this time. If you do not supply a username and password, you are prompted to supply a username and password when needed.
