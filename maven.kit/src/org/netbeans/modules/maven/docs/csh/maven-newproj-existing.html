New Maven Project Wizard: Project with Existing POM

See Also

The New Maven Project wizard enables you to create a new Maven project. You open the wizard by opening the New Project wizard and selecting the project template from the Maven category.

If you have a Maven project that has a POM you do not need to create a new project or perform any steps to import the project. To open the project in the IDE you can choose File > Open Project from the main menu and locate the project on your local system in the Open Project dialog box.

You can click Finish in this page of the wizard to open the Open Project dialog box.
