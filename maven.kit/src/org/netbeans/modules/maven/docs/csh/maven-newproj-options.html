New Maven Project Wizard: Module Options

See Also

The New Maven Project wizard enables you to create a new Maven project. You open the wizard by opening the New Project wizard and selecting the project template from the Maven category.

When you create a NetBeans application or module for a NetBeans application, you can use the Module Options page to set the following properties.

  • NetBeans Version. Select the version of the IDE that you want to use as the platform.
  • Allow OSGi bundles as Dependencies. When selected, you can use OSGi bundles as dependencies of the NetBeans Platform application or modules. This option is disabled if you select a version of the NetBeans platform that does not support OSGi.
  • Create Module Project. When selected, the IDE will also create a NetBeans Platform module and configure the module as a dependency of the application. You can specify the name of the module in the text field or use the default name. This option is only available when creating a Maven NetBeans Platform application.

When you click Finish, the project is created on your computer and opened in both the Projects window and the Files window.
