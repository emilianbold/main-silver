New Maven Project Wizard: Maven Archetype

See Also

The New Maven Project wizard enables you to create a new Maven project. You open the wizard by opening the New Project wizard and selecting Project from Archetype from the Maven category.

In the Maven Archetype page of the wizard you select an archetype to use as the template for your project from the Known Archetypes list. By default, older versions of archetypes are filtered out and only the most recent version of each known archetype is displayed. You can select the Show Older checkbox to remove the filter and view older versions of archetypes.

The Maven Archetype page of the wizard contains the following fields.

  • Filter. Type in the text field to limit the list of archetypes. When you type in the Filter field the list is updated to only display the archetypes that contain the entered text in the name, group ID, version, artifact ID, repository or description fields.
  • Show Older. When selected, older versions of archetypes are also included in the list. By default the list only contains the most recent version of the archetype.
  • Group ID. Displays the group ID of the selected archetype.
  • Version. Displays the version of the selected archetype.
  • Artifact ID. Displays the artifact ID of the selected archetype.
  • Repository. Displays the repository that contains the selected archetype.
  • Description. Displays a description of the selected archetype.

Click Next after you select the archetype to specify the project name and location.
