Heap Window

See Also

The Heap window is typically located in the main editor window of the IDE and displays detailed information about the heap when the was saved. You do not need to be running a profiling session to open or compare heap dumps.

You can open a heap snapshot in the Heap window by performing the following steps.

  • Choose Profile > Load Heap Dump in the main menu.
  • Locate the heap snapshot on your local system and click Open.

You can use the Profiler Control Panel to open saved heap snapshots that are associated with open projects.

You can click the link at the top of the window to compare two heap dumps and view the differences between the heap at two different points in time.
