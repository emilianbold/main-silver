Attach Profiler: Select Process to Attach

See Also

The Select Process to Attach dialog box enables you to attach the profiler to a process that is running on your local system. You see this dialog box when you use the Dynamic attach method to attach the profiler to a local application and you click Attach in the Attach Profiler dialog box.

To attach the profiler you need to select the process from the list of processes in the dropdown list. You can click Refresh to update the list of processes. When you select a process the dialog box displays details about the selected process such as the process ID (PID) and the main class. Click OK to attach the profiler to the selected process and start the profiling session.
