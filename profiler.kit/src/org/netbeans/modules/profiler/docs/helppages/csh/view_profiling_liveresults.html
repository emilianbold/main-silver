Live Results Window

See Also

The Live Results window displays the live profiling results when you are analyzing application performance or memory usage. No results are displayed if no profiling session is running or you are only monitoring the application.

You can open the Live Results window by choosing Window > Profiling > Live Results from the main menu. When you are analyzing application performance or memory usage you can click the Live Results button in the Profiler window to open the Live Results window .
