Select Profiling Task: Filter Value Dialog Box

See Also

The Filter Value dialog box displays the classes that will be be profiled when you run the profiling session with the selected filter.

You can access this window by selecting Show Filter Value in the Select Profiling Task dialog box when you select the Profile Only Project Classes or Profile Project and Subproject Classes filters.

You can create a custom QuickFilter of specific classes by clicking To QuickFilter. When you click To QuickFilter all the listed classes are added to the QuickFilter and the IDE opens the Set Quick Filter dialog box. In the Set Quick Filter dialog box you can choose the type of filter and modify the list of filtered classes by typing in the Filter Value text box. You can apply the filter to the profiling session by choosing Quick Filter in the Filter dropdown list in the Select Profiling Task window.
