OQL Console: Query Properties Dialog Box

See Also

The Query Properties dialog box enables you to modify the properties of custom saved OQL queries and modify the order that custom saved queries are listed under the Custom node in the Saved Queries pane.

You open the Query Properties dialog box by selecting a query in the Saved Queries pane of the OQL Console tab in the Heap window and clicking Properties.

  • You cannot modify the properties or change the order of the default saved queries that are bundled with the IDE and listed under the Samples node or the PermGen Analysis node.

The Query Properties dialog box contains the following fields.

  • Name. Modify the name of the query.
  • Description. (Optional) Modify the description of the query.

Click OK in the dialog box to save the modifications.

You can use the Up and Down arrows in the dialog box to modify the position of the selected custom query in the list under the Custom node.
