CPU Snapshot: Edit User Comments Dialog Box

See Also

The Edit User Comments dialog box enables you to add and edit comments that are saved in a CPU snapshot. You open this dialog by clicking the edit link in the User Comments section in the Info tab of a CPU snapshot. Type or modify the text in the text field and click OK to save the comments.
