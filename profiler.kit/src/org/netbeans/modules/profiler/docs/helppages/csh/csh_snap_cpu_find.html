CPU Snapshot: Find in Results Dialog Box

See Also

A CPU snapshot captures data on method call chains, times and invocations. You can take a CPU snapshot when you are analyzing application performance. The snapshot is displayed in the main editor window.

The Find in Results dialog box enables you to quickly search the results in the CPU snapshot window for a method or class. You can open the Find in Results dialog box by clicking the Find in Results button (  ) (Ctrl-F) in the toolbar of the following tabs in the CPU snapshot window:

  • Call Tree
  • Hot Spots
  • Combined

The Find in Results dialog box contains the following field:

  • Find What. Type the text in the field that you want to locate in the results in the window.
