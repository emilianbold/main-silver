Profiler Window: Rename Snapshot Dialog Box

See Also

The Rename Snapshot dialog box enables you to change the name of a saved snapshot or heap dump. You open this window by selecting a snapshot in the Saved Snapshots section of the Profiler Control Panel and clicking Rename.

To rename a snapshot you need to specify the following properties.

  • New File Name. Type the new name for the snapshot.
