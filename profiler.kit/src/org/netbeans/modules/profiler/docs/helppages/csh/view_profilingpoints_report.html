Profiling Points Report Window

See Also

The Profiling Points Report Window enables you to view details about the profiling point that you select in the Profiling Points window. You open this window by right-clicking a profiling point in the Profiling Points window and choosing Show Report in the popup menu.

The Profiling Points Report window displays the following details about the selected profiling point:

  • Type and status of profiling point
  • Location of profiling point, including the project and the line in the source code
  • Number of times the profiling point was hit
