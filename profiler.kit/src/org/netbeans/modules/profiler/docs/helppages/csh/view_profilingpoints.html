Profiling Points Window

See Also

The Profiling Points Window enables you to manage the profiling points in your projects. The Profiling Points Window is typically located below the editing area and sharing the space with the Output window. You open the Profiling Points Window by choosing Window > Profiling > Profiling Points.

The Profiling Points window displays the profiling points in open projects. You can use the Project dropdown list to limit the profiling points that are listed to a single project. You can select a profiling point and then use the toolbar to edit, remove, enable and disable the selected profiling point.

In the Profiling Points window you can right-click a profiling point and select the following commands in the popup menu:

  • Open the Profiling Points Report window that contains additional details about the selected profiling point
  • Open the source file in the editor at the location of the profiling point
  • Edit, disable, enable and delete the profiling point
