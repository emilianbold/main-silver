Select Heap Dump to Compare Dialog Box

See Also

The Select Heap Dump to Compare dialog box enables you to select a heap dump that is associated with the current project or select a heap dump that is located on your local system. The heap dump that you select will be compared to the heap dump in the Heap window.

You open this dialog box by choosing Compare with another heap dump at the top of the Classes tab in the Heap window.

In this dialog you specify the location of the heap dump by selecting one of the following options.

  • From project. When selected you can select a heap dump from the list of heap dumps that are associated with the current project.
  • From file. When selected you can click the Browse button to locate the heap dump that you want to compare to the current heap dump in the Heap window.

Click OK to view the differences between the specified heap and the heap dump that is open in the Heap window.
