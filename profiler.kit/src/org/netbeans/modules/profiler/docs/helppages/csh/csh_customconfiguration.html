Select Profiling Task: New Custom Configuration Dialog Box

See Also

The New Custom Configuration dialog box enables you to create a named set of options for a profiling task. You open this window by clicking Create Custom in the left pane of the Select Profiling Task dialog box when you select the profiling task.

A custom configuration can be based on the default settings for the task or you can base the new configuration on an existing custom configuration. A custom configuration enables you to modify the basic and advanced options for the profiling task.

To create a custom configuration you need to specify the following options.

  • Name. Type the name of the new custom configuration.
  • Initial Settings. Select a configuration on which the new configuration is to be based.

    Select Default if you want the new custom configuration to be based on the default configuration for the profiling task. Select From Existing Configuration to select a configuration from the list.
