New Profiling Point Wizard

See Also

The New Profiling Point wizard enables you to set a new profiling point that will invoke a profiling action. In this wizard you specify the type of profiling point that you want to set according to the profiling action that want to be invoked. Depending on the type of profiling point that you select, you will either specify a line in your code or you will specify a condition that needs to be met to invoke the action. In this wizard you will also specify settings for the action that is performed.

You open this wizard in the following ways.

  • Right-click in a line in your source code and choose Profiling > Insert Profiling Point.
  • Click Add Profiling Point in the toolbar of the Profiling Points window.

In this page of the wizard you specify the following properties.

  • Profiling Point Type. Select the type of profiling point according to the action that you want to invoke.
    • Reset Results
    • Stopwatch
    • Take Snapshot
    • Load Generator
    • Timed Take Snapshot
    • Triggered Take Snapshot

    A description of the type is displayed in the wizard when you select a type.

  • Profiling Point Project. Select a project from the dropdown list of open project.
