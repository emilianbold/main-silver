Select Profiling Task: Duplicate Configuration Dialog Box

See Also

The Duplicate Configuration dialog box enables you to create a copy of an existing configuration for a profiling task. In this dialog box you enter the new name of the configuration that you want to duplicate. You open this window by right-clicking the name of the configuration that you want to duplicate in the left pane of the Select Profiling Task dialog box and choosing Duplicate in the popup list. You can duplicate one of the default configurations or a custom configuration.

To duplicate a configuration you need to specify the following options.

  • Name. Type the name for the new copy of the selected configuration.
