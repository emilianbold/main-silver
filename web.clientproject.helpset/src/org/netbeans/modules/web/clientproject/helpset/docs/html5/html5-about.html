An HTML5 application project is a web application that is displayed in a browser on your desktop or a mobile device. An HTML 5 application is typically comprised of HTML, CSS and JavaScript files. The JavaScript that is used to manipulate and process objects in the application is generally handled on the client side by the JavaScript engine in the web browser. An HTML5 application is often used as a client that consumes various web services.

The IDE provides support for developing HTML5 applications, including the following tools.

  • Wizards for creating HTML5 applications and files
  • Support for JavaScript code-completion, debugging and testing
  • Tight web browser integration
  • CSS code completion
  • Generate JavaScript clients for RESTful web services
