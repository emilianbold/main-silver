HTML5 Application Tasks: Quick Reference

This topic describes common tasks you might want to perform when you are developing HTML5 applications. For more detailed information, click the links in the right column.

To perform this task Follow these steps
Create a project.
  1. Choose File > New Project (Ctrl-Shift-N) in the main menu.
  2. Select the correct project template in the New Project wizard.
  3. Step through the wizard to specify a site template option and add JavaScript files.
Add a JavaScript library to a project.
  1. In the Projects window, right-click the project node and choose Properties.
  2. Select the JavaScript Files category in the Project Properties window.
  3. Select the JavaScript file from the list in the left pane and click the right arrow button to move the selected file to the list in the right pane. Click OK.
Specify the target browser.
  1. Select the project node in the Projects window.
  2. Select the browser from the dropdown list in the Toolbar.

    Alternatively, right-click the project node and choose Properties in the popup menu and then select a browser from the dropdown list in the Run panel of the Project Properties window.
Inspect source code with the browser.
  1. Specify Chrome with NetBeans Integration as the target browser and install the NetBeans extension for Chrome.

    Alternatively, you can specify Embedded WebKit Browser.

  2. Run the project.
  3. Enable Inspect in NetBeans Mode.
View the application in various screen sizes.
  1. Specify either Chrome with NetBeans Integration or Embedded WebKit Browser as the target browser.
  2. Run the project.
  3. In the browser, select a screen size or create a custom screen size.
Run a project.
  1. Specify a target browser for the application.
  2. Right-click the project node in the Projects window and choose Run.
Debug a JavaScript file.
  1. Specify either Chrome with NetBeans Integration or Embedded WebKit Browser as the target browser.
  2. Set the breakpoints in the JavaScript file.
  3. Right-click the project node and choose Run.

    Alternatively, right-click the HTML file that invokes the JavaScript file and choose Run File.
Test a JavaScript file.
  1. In the Services window, right-click the JS Test Driver node and choose Configure.
  2. Configure JS Test Driver by specify the location of the JS Test Driver JAR and specifying the browsers that you would like to test the JavaScript against.
  3. Create a jsTest Driver Configuration File by choosing File > New File and selecting jsTest Driver Configuration File in the Unit Tests category.
  4. Create the JavaScript file and add the unit tests to the file.
  5. Edit the jsTest Driver configuration file to load the JavaScript files and the test files.
  6. Right-click the project node and choose Test.
