Running an HTML5 Application

See Also 

An HTML5 application project is typically run in a browser on your desktop or a mobile device. When you run an HTML5 application on your local system for testing or debugging you do not need to compile or deploy the application to a server because the JavaScript engine in the browser generally handles any scripts in the application. To run the project you only need to specify the browser and then run the application.

Specifying the browser

When you want to run an HTML5 application from the IDE you need to specify the target browser. The IDE enables you to easily select any of the browsers that are installed on your local system or you can select the embedded WebKit browser.

You can specify the target browser for the application in the following ways.

  • Select the application in the Projects window and then select the browser from the dropdown list in the Toolbar.
  • Open the Project Properties window and select the browser in the Run panel of the window.
  • If you want to debug your application or synchronize the IDE with your browser you should select either Chrome with NetBeans Integration or Embedded WebKit Browser as the target browser.

Running the project

After you specify the browser, you can run the application in the following ways.

  • Right-click the project node in the Projects window and choose Run in the popup menu.
  • Select the application in the Projects window and then click Run in the Toolbar or choose Run > Run Project (F6) in the main menu.

When you run an application or file the IDE opens the application or file in the selected browser.

  • If more than one project is open you can specify one of the projects as the Main Project. After you specify the project as the Main Project, the F6 keyboard shortcut will always run the Main Project regardless of the project that is selected in the Projects window.
  • You can run a single file by right-clicking the file in the Projects window and choosing Run File from the popup menu. Alternatively, you can select the file in the Projects window and choose Run > Run File from the main menu.
