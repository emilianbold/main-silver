HTML5: Customize Window Settings Dialog Box

See Also

When you want to preview an HTML5 application in the Embedded WebKit browser you can use this dialog box to customize the list of types of browser windows that you can select in the Web Browser toolbar. The dialog box displays a table that lists the screen sizes of web browsers that are used on various types of desktop and mobile devices.

You open this dialog box by clicking the Other Options icon () in the toolbar of the Web Browser window and choosing Customize in the dropdown list.

The icons for browser types in the toolbar of the Web Browser window are displayed in the order that they appear in the table. Select the checkbox in the Toolbar column in the table for each browser type that you want to appear in the toolbar. You can use the following buttons to the right of the table to manage the order of the browser types and to create and delete browser types.

  • New. Click New to create a custom browser type and size. When you click New a new row appears in the table. Type the new name and values directly in the text fields in the table.
  • Remove. Select a browser type from the list and click Remove to remove the type from the table.
  • Move Up and Move Down. Select a row in the table and use the Move Up and Move Down buttons to change the order of the icons in the Web Browser window toolbar.

You can modify the properties of a browser type by editing the text fields in the table. To edit a property, double-click in the text field of the property that you want to edit, type the new value and press the Return key on your keyboard. You can change the icon that is used to represent the browser type by clicking the icon in the Type column and choosing an icon from the dropdown menu.
