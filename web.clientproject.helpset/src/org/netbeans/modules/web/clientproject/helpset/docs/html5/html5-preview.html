Previewing an HTML5 Application

See Also 

An HTML5 application project is typically run in a browser on your desktop or a mobile device. The IDE can help you develop projects that can respond and adapt to different display devices by enabling you to preview pages at different screen sizes. When you run the HTML5 application from the IDE you can choose to view the pages in one of the default screen sizes or create a custom screen size.

To preview the project at different screen sizes you need to specify one of the following as the target browser.

  • Chrome with NetBeans Integration.
  • Embedded WebKit Browser.

Perform the following steps to select a screen size.

  1. Run the application to open the application in the target browser.
  2. In the browser, select a screen size from the menu.

    The location of the menu for selecting the screen size options depends upon the target browser.

    • Chrome with NetBeans Integration. Click the NetBeans icon in the URL location bar to open the menu and select a screen size.
    • Embedded WebKit Browser. Click the Other Options menu icon and select a screen size from the dropdown list.

      Alternatively, you can click a screen size icon in the toolbar.

    You can Select Customize in the menu to create custom sizes for windows and to edit the contents on the menu.

When you select a new screen size the display area of the browser resizes to the selected size. You can easily preview how the application responds to changes in the size of the display that might be specified in media queries in your style sheet.
