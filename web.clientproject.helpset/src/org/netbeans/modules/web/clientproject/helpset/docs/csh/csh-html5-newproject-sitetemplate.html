New HTML5 Application Wizard: Site Template

See Also 

The New HTML5 Application wizard creates a new project containing an empty HTML5 application. You open the New HTML5 Application wizard by opening the New Project wizard and selecting the HTML5 Application template from the HTML/JavaScript category.

In the Site Template page you need to select one of the following site template options.

  • No Site Template. Select this option to create a project. that will only contain an index.html file in the site root.
  • Select Archive. Select this option to create a project that is based on an archive that contains a site template. If you select this option you need to click Browse to locate the archive on your local system.
  • Download Online Template. Select this option to create a project that is based on an online template. If you select this option you need to select a template from the list.

Click Next to add JavaScript libraries to the project. You can click Finish if you do not need to add any JavaScript libraries.
