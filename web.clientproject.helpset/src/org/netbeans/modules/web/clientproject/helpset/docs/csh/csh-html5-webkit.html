HTML5: Web Browser Window

See Also

The Web Browser window displays an embedded web browser that is based on the WebKit browser engine. When you run an HTML5 applications in the Embedded WebKit Browser you can preview the application in the Web Browser window as it would appear on various desktop and mobile devices. You can enable Inspect mode to locate and navigate between an element that is selected in the browser and the corresponding source code.

The Web Browser window opens automatically when you run an HTML5 application and specify Embedded WebKit Browser as the target browser. You can open the Web Browser window by choosing Window > Web > Web Browser from the main menu.

The Web Browser window contains the following elements.

  • Location Bar. Displays the location of the current page. Pages are rendered using the WebKit browser engine running on the embedded lightweight server (localhost:8383). You can modify the server in the Project Properties window.
  • Toolbar. The toolbar contains the following elements.
    • Browser Types. Select a browser icon to preview the page contents in the selected browser type. When you select a browser type icon in the toolbar the viewable area for the web content is resized to the dimensions of the selected type.

      You can place your cursor over an icon to display the dimensions of the browser type. You can modify the browser icons that are displayed in the toolbar by choosing Customize in the Other Options dropdown list.

    • Other Options. Choose a browser type in the dropdown list to preview the page contents in the selected browser type.

      The dropdown list displays all available browser types. Choose Customize to open a dialog that enables you to modify, create or remove browser types.

    • Magnification. Type a value in the text field or select a value in the dropdown list to change the magnification of the contents of the page.
    • Inspect. Select to enable Inspect mode in the page.
