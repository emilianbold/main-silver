Debugging an HTML5 Application Project

See Also 

To debug an HTML5 application in the IDE the IDE needs to be able to communicate with the debugger. The application is automatically run in the debugger if you select either of the following browsers as the target browser in the project Properties window.

  • Embedded WebKit Browser
  • Chrome browser with the NetBeans extension
