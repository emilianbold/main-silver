Create Site Template from Current Project Dialog Box

See Also 

The Create Site Template from Current Project dialog box enables you to creates a site template .zip archive that is based on your HTML5 project. You open the dialog box by right-clicking the HTML5 project node in the Projects window and selecting Save as Template in the popup menu.

In the Describe Template panel of the Create Site Template from Current Project dialog box you need to specify the following properties.

  • Name. Specifies the name of the site template archive.
  • Destination. Specifies the location where you want to save the site template archive.
  • Files to Include. Expand the tree and select the files that you want to include in the site template.

Click Finish to create a .zip archive of the site template.
