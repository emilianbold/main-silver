Creating a New Project

See Also

If you want to create a new application in the IDE, you need to first have a project to work in. You can open an existing project or create a new project.

When you create a project, you have your choice between the following groups of templates:

  • BlueJ. Contains the Empty BlueJ Project template, which enables you to create a project that can be opened in both NetBeans IDE and BlueJ. Projects created with this template do not contain the full set of NetBeans IDE project features, but you can later convert such a project into a NetBeans IDE project.
  • General. Contains several different project templates for creating Java SE projects. When you select a template, a description of that template appears in the bottom panel. If you are creating a new application, the Java Application template is the most useful template.

    When you use one of these templates, your project has the full range of NetBeans IDE project features, but the project can not be opened in BlueJ or in the IDE's BlueJ View window.

  • Samples. This folder contains a sample application and a sample project that contains examples of some Swing user interfaces.

To create a new IDE project:

  1. Choose File > New Project.
  2. In the wizard, select a template and complete the wizard.
