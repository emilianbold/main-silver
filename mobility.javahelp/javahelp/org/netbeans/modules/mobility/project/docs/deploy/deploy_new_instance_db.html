Mobility: New Deployment Instance Dialog Box

See Also

Use this dialog box to create an instance of a deployment type (for example, an FTP server connection).

The New Deployment Instance dialog box appears after you have clicked the Add Instance button in the Mobility Deployment dialog box.

In the New Deployment Instance dialog box you have to specify the following:

  • Select Deployment Type. Select a method for deploying your MIDlet suite files to another location on your system or on a remote network.
  • New Instance Name. Enter a name for a new instance of the selected deployment type.
    • You must specify a valid Java identifier as a name (for example, the name must begin with a Unicode letter, the underscore character ( _ ), or a dollar sign ( $ ).
  • You can customize project-specific deployment options on the Deploying properties page of a project.

After you click OK, the new instance appears in the Deployment Types/Instances list of the Mobility Deployment dialog box.
