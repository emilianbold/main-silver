Add Cloud Provider Wizard: Oracle Cloud

See Also 

This panel of the wizard enables you to enter the details of the Oracle Cloud account that you want to use with the IDE. You see this panel when you select Oracle Cloud in the Add Cloud Provider wizard. You open the Add Cloud Provider wizard by right-clicking the Cloud node in the Services window and selecting Add Cloud. Alternatively, you can choose Tools > Cloud Providers from the main menu and click Add Cloud in the Cloud Providers manager.

To register Oracle Cloud with the IDE, enter the following information:

  • Data Center. Select the data center from the drop-down list.
  • Identity Domain. The individual or group identity of your Oracle Cloud account. If you do not have an Oracle Cloud account, click Open a New Account.
  • Java Service Name. The name of the Java Service you created when you set up your Oracle Cloud account.
  • Database Service Name. The name of the Database Service you created when you set up your Oracle Cloud account.
  • Service Administrator. Your identity as Oracle Cloud administrator (the email you gave when you set up your Oracle Cloud account).
  • Password. Your Oracle Cloud administrator password.
  • SDK. Path to your local copy of the Oracle Cloud SDK. Click Configure to browse for this file.
