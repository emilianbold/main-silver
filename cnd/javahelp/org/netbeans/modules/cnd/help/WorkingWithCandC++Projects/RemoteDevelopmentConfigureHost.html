Configuring a Remote Host for C/C++/Fortran Development

See Also 

You can configure a remote host to use for building, running, and debugging your C, C++, and Fortran projects. The remote host must meet the following conditions:

  • Remote host operating system must be Solaris, OpenSolaris, or Linux.
  • Remote host must run the Secure Shell (SSH) daemon, sshd.
  • At least one tool collection (GNU, Oracle Solaris Studio, or Sun Studio) must be accessible on the remote host
  • You must have a user account on the host.

To configure a remote host:

  1. Select Tools > Options, and click the C/C++ category.
  2. Click Build Tools.
  3. Click the Edit button next to the Development Host list to open the Development Host Manager dialog box.
  4. In the Development Host Manager dialog box, click Add. The New Remote Development Host wizard starts to guide you through adding the remote host. Click the Help button in the wizard for more information.

You can also add a development host from the Services window. Select Window > Services to open the window, then right-click the C/C++ Development Hosts node in the Services window, and select Add New Host.
