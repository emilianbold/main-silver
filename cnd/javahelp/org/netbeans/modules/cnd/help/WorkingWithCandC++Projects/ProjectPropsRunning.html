C/C++ Project Properties Dialog Box: Run

See Also 

The Run panel lets you specify properties that are used when you run your project in the IDE.

Arguments. Enter any arguments that should be supplied to the project's command line when you run the project.

Run directory. Enter the path to the directory where the project should run. Normally this is the same as the project directory.

Environment. Enter any environment variables and their values if the project requires them to run.

Build First. Select the checkbox if you want the project to build before it runs.

Console Type. The output generated when you run the project goes to the console. You can specify what type of console to use. Select Output Window to send the program's output to the Output window in the IDE. This causes the build output to be replaced with the run output. Select External Terminal to cause the IDE to open a new terminal window outside the IDE to display the program output and allow you to type input to the program. Select Internal Terminal to cause the IDE to open a new window in the IDE, separate from the Output window, to display the program output and allow you to type any input.

Terminal Type. The terminal type applies to the External Terminal. The choices offered in the list vary according to the environment where the IDE is running.
