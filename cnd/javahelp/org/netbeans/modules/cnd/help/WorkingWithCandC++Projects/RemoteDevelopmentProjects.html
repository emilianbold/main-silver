Setting Up C/C++/Fortran Projects for Remote Development

See Also 

After you have configured a remote buid host, you can select the remote host to build and run a project.

To set up an existing project to use a remote build host:

  • In the Projects window, right-click the project, choose Set Build Host, and select a remote host.

To set up a new project to use a remote build host:

  • In the New Project wizard's Project Name and Location panel, select the host from the Build Host list.
