Setting Up C/C++/Fortran Projects for Remote Development

See Also 

After you have configured a remote development host, you can set up a project to use the remote host to build and run the project.

To set up an existing project to use a remote development host:

  • In the Projects window, right-click the project and select Set Remote Development Host > user@server.

To set up a new project to use a remote development host:

  • In the project wizard's Project Name and Location panel, select the host from the Development Host list.
