C/C++ Project Properties Dialog Box: Profile

See Also 

The Profile panel enables you to specify properties of the profiling tools that are displayed in the Run Monitor window when C/C++ projects are run.

  • Show profile indicators during run: Select this option to show C/C++ profiling tools in the Run Monitor window whenever the project runs. The Profile Configuration determines which of the tools are shown.
  • Profile Configuration: Profile Configurations are sets of profiling tools. The Profile Configurations shown in the drop down list depend on your operating system and tool collection.
    • C/C++ DTrace configurations are only supported on a Solaris OS and do not require the Sun Studio tool collection.
    • C/C++ Sun Studio configurations are supported on Solaris OS and Linux systems that have the Sun Studio tool collection.
    • C/C++ Simple configurations provide the graphic indicators without the deeper analysis for detailed information and can be used on Solais OS or Linux, without Sun Studio tool collection.
    Click the button next to the Profile Configuration list to open the Profiler Tools Manager dialog box, where you can see which tools are included in each Profile Configuration.
