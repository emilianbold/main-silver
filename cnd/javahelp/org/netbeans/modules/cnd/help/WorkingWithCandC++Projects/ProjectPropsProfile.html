C/C++/Fortran Project Properties Dialog Box: Profile

See Also 

The Profile panel enables you to specify properties of the profiling tools that are displayed in the Run Monitor window when C/C++/Fortran projects are run.

  • Show profiling indicators during run: Select this option to show C/C++ profiling tools in the Run Monitor window whenever the project runs. The Profile Configuration determines which of the tools are shown.
  • Profile Configuration: Profile Configurations are sets of profiling tools. The Profile Configurations shown in the drop down list depend on your operating system and tool collection.
    • C/C++ DTrace configurations are only supported on an Oracle Solaris operating system but do not require the Oracle Solaris Studio tool collection.
    • C/C++ Oracle Solaris Studio configurations are supported on Oracle Solaris and Linux systems that have the Oracle Solaris Studio tool collection.
    • C/C++ Simple configurations provide the graphic indicators without the deeper analysis for detailed information and can be used on Oracle Solaris or Linux, without Oracle Solaris Studio tool collection.
    Click the button next to the Profile Configuration list to open the Profiler Tools Manager dialog box, where you can see which tools are included in each Profile Configuration. See Managing C/C++/Fortran Profile Configurations for more information.
