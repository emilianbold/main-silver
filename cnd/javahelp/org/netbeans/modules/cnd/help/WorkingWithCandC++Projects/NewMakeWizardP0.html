New C/C++ Project With Existing Sources Wizard: Select Mode

See Also 

The New C/C++ Project With Existing Sources wizard enables you to create a project using an existing makefile.

In the Select Mode page, you specify how the IDE should configure the project.

  • Automatic. Lets the IDE automatically configure the project based on an existing makefile or configure script in the source folder. Note that when the IDE configures the project automatically, it does a clean and build after creating the project. You should not choose Automatic if you have built binaries in your source folder. that you want to keep.
  • Custom. Lets you determine how the project is configured.
