Debugging a C or C++ Unit Test

***this will need to be updated when c unit test implementation complete

See Also

  • Right-click the folder for the test and select Debug Test.

    The IDE runs the test in the debugger, enabling you to examine your tests for errors.

  • You can also step into the test by right-clicking the folder for the test and selecting Step Into.
