Unit Test Results Window

See Also

Unit test results are displayed in the Test Results window. When you run a test, the IDE automatically opens the window and displays the results in two tabs:

  • A summary of the passed and failed tests and the description of failed tests are displayed in the left pane of the window. Click the filter button (  ) to switch between viewing all test results and viewing only the failed tests.
  • The textual output from the tests is displayed in the right pane of the window.

After you run a test you can right-click any test displayed in the Test Results and choose Run Again from the popup menu to run the individual test again.

You can also double-click a test to go to the source for the test.
